曲面积分 第一型 vs 第二型

一句话：曲面积分是"把函数累加在曲面上"，第一型只管面积不管方向，第二型必须盯着曲面的侧。
这一节是格林公式在三维空间的自然延伸，也是高斯公式的直接铺垫。

💡 一句话本质

第一型：在曲面上"称质量"（只有大小没有方向）
第二型：穿过曲面的"流量"（有进有出，方向决定正负）

📖 为什么需要曲面积分

格林公式把平面上的曲线积分转化成了二重积分。自然的问题是：三维空间中的曲面积分，能不能也有类似的转化？

答案是高斯公式（下一节），但它需要你先学会计算曲面积分本身。所以本节是"工具"——先学会把曲面积分化成二重积分算出来，下一节再用高斯公式巧算。

类比理解

曲线积分是"沿着一条线"累加；曲面积分是"铺在一片面上"累加。第一型相当于给曲面"称总重"，第二型相当于看流体"穿过了多少"。

📐 第一型曲面积分（对面积的积分）

定义与记号

\[\iint_\Sigma f(x,y,z)\,dS\]

其中 \(\Sigma\) 是一张光滑曲面，\(dS\) 是面积元素。第一型曲面积分与曲面的方向无关——无论你把曲面翻到哪一面，积分值都一样。

物理意义

如果 \(f(x,y,z)\) 是曲面薄片的面密度，那么 \(f\,dS\) 就是一小块面积的质量，整个积分就是这张曲面的总质量。

核心计算公式

第一型的计算思想永远是一样的：投影到坐标面，把面积元素用偏导数表示出来。

曲面方程投影区域面积元素 \(dS\)积分转化
\(z = z(x,y)\) \(D_{xy}\) \(\sqrt{1+z_x^2+z_y^2}\,dxdy\) \(\displaystyle\iint_{D_{xy}} f(x,y,z(x,y))\sqrt{1+z_x^2+z_y^2}\,dxdy\)
\(y = y(x,z)\) \(D_{xz}\) \(\sqrt{1+y_x^2+y_z^2}\,dxdz\) \(\displaystyle\iint_{D_{xz}} f(x,y(x,z),z)\sqrt{1+y_x^2+y_z^2}\,dxdz\)
\(x = x(y,z)\) \(D_{yz}\) \(\sqrt{1+x_y^2+x_z^2}\,dydz\) \(\displaystyle\iint_{D_{yz}} f(x(y,z),y,z)\sqrt{1+x_y^2+x_z^2}\,dydz\)
考场上最常考的情况

90% 的题曲面方程都写成 \(z = z(x,y)\)，投影到 \(xOy\) 面计算。所以你最需要背熟的是：

\[dS = \sqrt{1+\left(\frac{\partial z}{\partial x}\right)^2+\left(\frac{\partial z}{\partial y}\right)^2}\,dxdy\]

标准计算步骤（投影到 \(xOy\) 面）

1

把曲面写成 \(z = z(x,y)\) 的显式方程

如果是隐式方程（如球面 \(x^2+y^2+z^2=R^2\)），通常取上半球 \(z=\sqrt{R^2-x^2-y^2}\) 或下半球。

2

确定投影区域 \(D_{xy}\)

把曲面方程中的 \(z\) 消去，得到在 \(xOy\) 面上的投影范围。球面投影就是圆，锥面投影也是圆。

3

计算偏导数和面积元素

求 \(z_x, z_y\)，代入 \(dS = \sqrt{1+z_x^2+z_y^2}\,dxdy\)。

4

代入被积函数，化为二重积分

把 \(z\) 换成 \(z(x,y)\)，所有量都变成 \(x,y\) 的函数，然后在 \(D_{xy}\) 上算二重积分。

对称性（第一型可以放心用）

第一型曲面积分的对称性和三重积分完全类似，因为 \(dS > 0\) 恒成立，没有方向带来的符号问题。

对称性口诀

若曲面 \(\Sigma\) 关于 \(xOy\) 面对称（上下对称）：

  • 被积函数 \(f(x,y,z)\) 关于 \(z\) 是奇函数 → 积分 = 0
  • 被积函数 \(f(x,y,z)\) 关于 \(z\) 是偶函数 → 积分 = 2倍上半部分

关于 \(yOz\) 面、\(xOz\) 面的对称性同理。

重要提醒

第一型的对称性和第二型完全不同！第二型因为有方向，奇函数反而可能是2倍、偶函数反而可能是0。下一节细讲。

📐 第二型曲面积分（对坐标的积分）

定义与记号

\[\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy\]

也可以写成向量形式：\(\displaystyle\iint_\Sigma \vec{F}\cdot d\vec{S}\)，其中 \(\vec{F}=(P,Q,R)\)，\(d\vec{S}\) 是有向面积元素。

物理意义：流量

想象 \(\vec{F}\) 是一个流速场，\(\Sigma\) 是一张有方向的膜。第二型曲面积分就是单位时间内从指定方向穿过这张膜的流体净体积。如果流体从反方向穿，积分值为负。

曲面的"侧"——方向从哪里来？

第二型曲面积分必须指定曲面的侧。一张曲面有两个面，你必须告诉积分"从哪一侧穿过去算正"。

投影分量方向名称法向量分量特征符号
\(dxdy\) 上侧 / 下侧 法向量的 \(z\) 分量：\(\cos\gamma > 0\) 为上侧 上正下负
\(dydz\) 前侧 / 后侧 法向量的 \(x\) 分量：\(\cos\alpha > 0\) 为前侧 前正后负
\(dzdx\) 右侧 / 左侧 法向量的 \(y\) 分量：\(\cos\beta > 0\) 为右侧 右正左负
死记这个口诀

上正下负，前正后负，右正左负

这里的"上"是 \(z\) 轴正向，"前"是 \(x\) 轴正向，"右"是 \(y\) 轴正向。考试紧张时默念三遍。

投影法计算第二型

第二型的计算可以拆成三个独立积分：

\[\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \iint_\Sigma P\,dydz + \iint_\Sigma Q\,dzdx + \iint_\Sigma R\,dxdy\]

每一项分别投影到对应的坐标面：

1

计算 \(\displaystyle\iint_\Sigma R\,dxdy\)

把曲面写成 \(z = z(x,y)\)，投影到 \(D_{xy}\)。
上侧：\(\displaystyle+\iint_{D_{xy}} R(x,y,z(x,y))\,dxdy\)
下侧：\(\displaystyle-\iint_{D_{xy}} R(x,y,z(x,y))\,dxdy\)

2

计算 \(\displaystyle\iint_\Sigma P\,dydz\)

把曲面写成 \(x = x(y,z)\)，投影到 \(D_{yz}\)。
前侧（\(x\) 大的一侧）：\(\displaystyle+\iint_{D_{yz}} P(x(y,z),y,z)\,dydz\)
后侧：\(\displaystyle-\iint_{D_{yz}} P(x(y,z),y,z)\,dydz\)

3

计算 \(\displaystyle\iint_\Sigma Q\,dzdx\)

把曲面写成 \(y = y(x,z)\)，投影到 \(D_{xz}\)。
右侧（\(y\) 大的一侧）：\(\displaystyle+\iint_{D_{xz}} Q(x,y(x,z),z)\,dzdx\)
左侧：\(\displaystyle-\iint_{D_{xz}} Q(x,y(x,z),z)\,dzdx\)

经典错误

投影法中，投影区域必须是被积函数曲面对应坐标面的投影。例如计算 \(R\,dxdy\) 时，曲面方程必须解出 \(z\) 作为 \(x,y\) 的函数，投影到 \(xOy\) 面。不能张冠李戴。

第二型的对称性——陷阱警告

第二型曲面积分的对称性和第一型完全相反，原因是 \(dydz, dzdx, dxdy\) 本身带有方向（可正可负）。

核心陷阱

若 \(\Sigma\) 关于 \(xOy\) 面对称（上下对称），考虑 \(R\,dxdy\) 项：

  • \(R(x,y,z)\) 关于 \(z\) 是奇函数 → \(R\,dxdy\) 项 = 2倍上半部分（不是0！）
  • \(R(x,y,z)\) 关于 \(z\) 是偶函数 → \(R\,dxdy\) 项 = 0（不是2倍！）

为什么？ 上下两侧的法向量 \(z\) 分量相反，所以 \(dxdy\) 的符号相反；奇函数本身也反号，负负得正，两个对称点贡献同向叠加。

安全策略

如果你对称性不熟，第二型曲面积分一律不用对称性，直接投影计算。考场上算慢一点也比猜错对称性丢分强。

📐 合一投影法

投影法的痛点是：三个分量要分别投到三个不同的坐标面，计算量大。如果曲面方程是 \(z = z(x,y)\)，我们可以把三项都投影到 同一个 \(xOy\) 面 上，这就是合一投影法。

推导思路

对于曲面 \(z = z(x,y)\)，面积元素之间的关系由法向量决定。上侧的单位法向量：

\[(\cos\alpha, \cos\beta, \cos\gamma) = \frac{1}{\sqrt{1+z_x^2+z_y^2}}(-z_x, -z_y, 1)\]

由两类曲面积分的关系 \(dydz = \cos\alpha\,dS\)，\(dzdx = \cos\beta\,dS\)，\(dxdy = \cos\gamma\,dS\)，以及 \(dS = \sqrt{1+z_x^2+z_y^2}\,dxdy\)，可得：

\[dydz = -z_x\,dxdy, \quad dzdx = -z_y\,dxdy\]

合一投影公式

合一投影法（投影到 xOy 面）

若 \(\Sigma: z = z(x,y)\)，\((x,y)\in D_{xy}\)，则

\[\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \pm\iint_{D_{xy}}\Big[-P\cdot z_x - Q\cdot z_y + R\Big]\,dxdy\]

上侧取正号，下侧取负号。（式中 \(P,Q,R\) 的 \(z\) 都要代成 \(z(x,y)\)。）

记忆技巧

把 \(dydz, dzdx, dxdy\) 都换成 \(dxdy\) 的"价格"：
\(dydz = -z_x\,dxdy\)（对 \(x\) 求偏导，带负号）
\(dzdx = -z_y\,dxdy\)（对 \(y\) 求偏导，带负号）
\(dxdy = 1\cdot dxdy\)（不变）

什么时候用合一投影法？
  • 曲面方程明确写成 \(z = z(x,y)\)（或可以方便地解出来）
  • 被积函数同时有 \(P, Q, R\) 三个分量
  • 投影到 \(xOy\) 面的区域比较简单（圆、矩形等）

如果曲面是垂直于 \(xOy\) 面的柱面（如 \(x^2+y^2=a^2\)），则投影到 \(xOy\) 是一条线，面积为零，此时 \(R\,dxdy = 0\)，但 \(P, Q\) 项仍需另算。

🔗 两类曲面积分的联系

两类曲面积分之间的桥梁是单位法向量。

\[\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \iint_\Sigma (P\cos\alpha + Q\cos\beta + R\cos\gamma)\,dS\]

其中 \((\cos\alpha, \cos\beta, \cos\gamma) = \vec{n}^0\) 是曲面 指定侧 的单位法向量。

什么时候用这个联系？

当你想把第二型转化成第一型算（利用第一型的对称性），或者反过来把第一型写成第二型再用高斯公式。

📋 第一型 vs 第二型 对照表

对比项第一型 \(\displaystyle\iint_\Sigma f\,dS\)第二型 \(\displaystyle\iint_\Sigma P\,dydz+Q\,dzdx+R\,dxdy\)
有无方向无，与侧无关有，必须指定侧
面积元素\(dS > 0\)\(dydz, dzdx, dxdy\) 可正可负
物理意义曲面质量穿过曲面的流量
对称性和三重积分一样（奇零偶倍）相反（奇倍偶零）
计算方法投影 + 算 \(dS\)投影 + 注意正负号 / 合一投影法
能否用高斯需先转成第二型可以直接用

🔗 与前后知识的联系

知识网络
  • 向前：格林公式处理平面曲线积分；曲面积分是三维空间的自然延伸，数学结构非常相似（都是"在几何对象上累加函数"）。
  • 向后：高斯公式（\(\text{div}\vec{F}\) 的三重积分 = 封闭曲面外侧的第二型曲面积分）直接把第二型曲面积分的计算难度降了一个维度——本节学投影法，下节学高斯巧算。
  • 横向：两类曲面积分的关系 \(\vec{F}\cdot d\vec{S}\) 和两类曲线积分的关系 \(\vec{F}\cdot d\vec{r}\) 是完全对应的向量思想。
