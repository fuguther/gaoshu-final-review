10.2 · 题型

幂级数 三类题型标准步骤

考试只考三件事：求收敛域、求和函数、把函数展开成幂级数。
步骤最固定，练熟就能拿满。

本页导航

  • 三步定题型
  • 题型一：求收敛域
  • 题型二：求和函数
  • 题型三：函数展开
  • 常见错误

🎯 三步定题型

Step 1：看题目问什么
├─ 给级数，问"收敛半径/收敛域" → 题型一
├─ 给级数，问"和函数" → 题型二
└─ 给函数，问"展开成幂级数" → 题型三
Step 2：题型二和题型三，先看能否匹配五个"招牌"
└─ 变形后一定能套 \(\frac{1}{1-x}, e^x, \sin x, \cos x, \ln(1+x)\)
Step 3：所有题，最后一步都是写收敛域
└─ 收敛域不写 = 扣分，端点不单独判断 = 扣分
考场口诀

"给级数求域用比值，给级数求和看 n 的位置，给函数展开找招牌；端点必须单独判，收敛域必须写。"

题型一：求收敛半径与收敛域

标准步骤

1

求收敛半径

标准幂级数 \(\sum a_n x^n\)：用比值法 \(R = \lim\limits_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right|\)。
缺项级数（如只含偶次项）：令 \(t = x^k\) 化为标准形，或直接用根值法。

2

写出收敛区间

得到开区间 \((-R, R)\)（若中心在 \(x_0\)，则为 \((x_0-R, x_0+R)\)）。

3

单独判断端点

代入 \(x = R\) 和 \(x = -R\)（或 \(x_0\pm R\)），用数项级数方法判敛散。

4

写出收敛域

四种可能：\((-R,R),\;[-R,R),\;(-R,R],\;[-R,R]\)。

例题 1 收敛域 · 经典调和/交错级数
求幂级数 \(\displaystyle\sum_{n=1}^{\infty} \frac{x^n}{n}\) 的收敛域。
✓ 标准解答

Step 1　系数 \(a_n = \frac{1}{n}\)，用比值法求收敛半径：

\[R = \lim_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right| = \lim_{n\to\infty}\frac{1/n}{1/(n+1)} = \lim_{n\to\infty}\frac{n+1}{n} = 1\]

Step 2　收敛区间为 \((-1, 1)\)。

Step 3　单独判断端点：

当 \(x = 1\) 时，级数化为 \(\sum\limits_{n=1}^{\infty}\frac{1}{n}\)，为调和级数，发散。

当 \(x = -1\) 时，级数化为 \(\sum\limits_{n=1}^{\infty}\frac{(-1)^n}{n}\)，为交错级数，满足莱布尼茨条件，收敛。

Step 4　综合得收敛域：

答案：\(\boxed{[-1,\;1)}\)

例题 2 收敛域 · 中心偏移型
求幂级数 \(\displaystyle\sum_{n=1}^{\infty} \frac{(x-1)^n}{n\cdot 2^n}\) 的收敛域。
✓ 标准解答

Step 1　令 \(t = x-1\)，级数化为 \(\sum\limits_{n=1}^{\infty}\frac{t^n}{n\cdot 2^n}\)。

系数 \(a_n = \frac{1}{n\cdot 2^n}\)，用比值法：

\[R_t = \lim_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right| = \lim_{n\to\infty}\frac{(n+1)\cdot 2^{n+1}}{n\cdot 2^n} = \lim_{n\to\infty}\frac{n+1}{n}\cdot 2 = 2\]

故 \(|t| < 2\)，即 \(|x-1| < 2\)，收敛区间为 \((-1, 3)\)。

Step 2　单独判断端点：

当 \(x = 3\)（即 \(t = 2\)）时，级数化为 \(\sum\limits_{n=1}^{\infty}\frac{2^n}{n\cdot 2^n} = \sum\limits_{n=1}^{\infty}\frac{1}{n}\)，调和级数，发散。

当 \(x = -1\)（即 \(t = -2\)）时，级数化为 \(\sum\limits_{n=1}^{\infty}\frac{(-2)^n}{n\cdot 2^n} = \sum\limits_{n=1}^{\infty}\frac{(-1)^n}{n}\)，交错级数，收敛。

Step 3　综合得收敛域：

答案：\(\boxed{[-1,\;3)}\)

技巧提炼

中心在 \(x=x_0\) 的幂级数，求收敛域的步骤与标准形完全一致：先求关于 \(t=x-x_0\) 的收敛半径，再换回去，最后别忘了端点也要换回 \(x\) 的值。

题型二：求幂级数的和函数

核心口诀

"分母有 n → 逐项求导消掉；分子有 n → 逐项积分消掉。"
目标永远是把级数变成等比级数 \(\sum x^n = \frac{1}{1-x}\)（\(|x|<1\)）。

标准步骤

1

设 \(S(x)\)，先求收敛域

和函数必须在收敛域内才有意义，端点处的和函数值有时可通过连续性单独讨论。

2

看 n 的位置，选择求导或积分

分母含 \(n\)（如 \(\frac{x^n}{n}\)）→ 逐项求导消掉分母；
分子含 \(n\)（如 \(n x^n\)）→ 逐项积分消掉分子。

3

化简为等比级数，求和

套 \(\sum x^n = \frac{1}{1-x}\)（\(|x|<1\)）。

4

逆操作还原

求导后积分回来，积分后求导回来。注意积分常数 \(C\) 的确定。

5

代入特殊点（通常是 \(x=0\)）定常数 \(C\)

定 \(C\) 时一定要回到原级数看 \(S(0)\)，不要代入求导后的式子。

例题 3 和函数 · 分子有 n（先积后导）
求幂级数 \(\displaystyle\sum_{n=1}^{\infty} n x^{n-1}\) 的和函数，并写出收敛域。
✓ 标准解答

Step 1　先求收敛域。系数 \(a_n = n\)，

\[R = \lim_{n\to\infty}\frac{n}{n+1} = 1\]

端点 \(x = \pm 1\) 时通项 \(n(\pm 1)^{n-1}\) 不趋于 0，发散。故收敛域为 \((-1, 1)\)。

设和函数 \(S(x) = \sum\limits_{n=1}^{\infty} n x^{n-1}\)（\(|x|<1\)）。

Step 2　分子有 \(n\)，选择先积分：

\[\int_0^x S(t)\,dt = \sum_{n=1}^{\infty}\int_0^x n t^{n-1}\,dt = \sum_{n=1}^{\infty} x^n = \frac{x}{1-x}\]

（最后一步利用了等比级数求和，\(|x|<1\)。）

Step 3　两边求导还原：

\[S(x) = \left(\frac{x}{1-x}\right)' = \frac{(1-x)\cdot 1 - x\cdot(-1)}{(1-x)^2} = \frac{1}{(1-x)^2}\]

Step 4　验证 \(x=0\)：原级数 \(S(0) = 1\)，公式给出 \(\frac{1}{(1-0)^2}=1\)，一致。

答案：\(\boxed{S(x) = \dfrac{1}{(1-x)^2},\quad x\in(-1,\;1)}\)

例题 4 和函数 · 分母有 n（先导后积）
求幂级数 \(\displaystyle\sum_{n=1}^{\infty} \frac{x^n}{n}\) 的和函数，并写出收敛域。
✓ 标准解答

Step 1　先求收敛域。由例题 1 已知收敛域为 \([-1, 1)\)。

设 \(S(x) = \sum\limits_{n=1}^{\infty}\frac{x^n}{n}\)（\(x\in[-1,1)\)）。

Step 2　分母有 \(n\)，选择先求导：

\[S'(x) = \sum_{n=1}^{\infty}\frac{n x^{n-1}}{n} = \sum_{n=1}^{\infty} x^{n-1} = \frac{1}{1-x}\]

（\(|x|<1\) 时成立。）

Step 3　两边从 \(0\) 到 \(x\) 积分还原：

\[S(x) - S(0) = \int_0^x \frac{1}{1-t}\,dt = -\ln|1-x|\]

Step 4　定常数。回到原级数，\(S(0) = 0\)（首项为 \(\frac{0^1}{1}=0\)，且 \(n\geq 1\) 时每一项都含 \(x\) 的正幂次）。

又因为在收敛域内 \(x < 1\)，所以 \(1-x > 0\)，可去掉绝对值：

\[S(x) = -\ln(1-x)\]

Step 5　验证端点 \(x=-1\)：原级数 \(\sum\frac{(-1)^n}{n} = -\ln 2\)，公式给出 \(-\ln(1-(-1)) = -\ln 2\)，一致。

答案：\(\boxed{S(x) = -\ln(1-x),\quad x\in[-1,\;1)}\)

和函数的核心策略

分子含 \(n\) 时，积分会把 \(n\) 消掉（\(\int n x^{n-1} = x^n\)）；分母含 \(n\) 时，求导会把 \(n\) 消掉（\((x^n/n)' = x^{n-1}\)）。逆操作时，积分常数一定要通过原级数在特殊点的值来确定，绝不能代入求导后的式子。

题型三：把函数展开成幂级数

标准步骤

1

识别"招牌"

看函数里有没有 \(e^{\square}, \sin\square, \cos\square, \ln(1+\square), \frac{1}{1-\square}\) 等熟悉形式。

2

变形匹配

通过换元、提常数、部分分式分解、求导/积分等操作，把函数变成 5 个基本展开式的标准形式。

3

代入公式，写出展开式

写出 \(\sum a_n x^n\) 的形式，并注明收敛域。

4

确定收敛域

收敛域取各展开式收敛区间的交集，端点需单独验证。

例题 5 函数展开 · 提常数配等比
将 \(f(x) = \dfrac{1}{2+x}\) 在 \(x=0\) 处展开成幂级数，并写出收敛域。
✓ 标准解答

Step 1　变形，凑 \(\frac{1}{1-t}\) 的标准形式：

\[f(x) = \frac{1}{2+x} = \frac{1}{2}\cdot\frac{1}{1+\frac{x}{2}} = \frac{1}{2}\cdot\frac{1}{1-\left(-\frac{x}{2}\right)}\]

Step 2　令 \(t = -\frac{x}{2}\)，利用 \(\frac{1}{1-t} = \sum\limits_{n=0}^{\infty} t^n\)（\(|t|<1\)）：

\[\frac{1}{2+x} = \frac{1}{2}\sum_{n=0}^{\infty}\left(-\frac{x}{2}\right)^n = \sum_{n=0}^{\infty}\frac{(-1)^n}{2^{n+1}}x^n\]

Step 3　确定收敛域。由 \(|t| = \left|-\frac{x}{2}\right| < 1\) 得 \(|x| < 2\)。

端点 \(x = 2\)：通项 \(\frac{(-1)^n}{2^{n+1}}\cdot 2^n = \frac{(-1)^n}{2}\) 不趋于 0，发散。

端点 \(x = -2\)：通项 \(\frac{1}{2^{n+1}}\cdot (-2)^n = \frac{(-1)^n}{2}\) 不趋于 0，发散。

答案：\(\boxed{\displaystyle\frac{1}{2+x} = \sum_{n=0}^{\infty}\frac{(-1)^n}{2^{n+1}}x^n,\quad x\in(-2,\;2)}\)

例题 6 函数展开 · 部分分式分解
将 \(f(x) = \dfrac{1}{x^2-3x+2}\) 在 \(x=0\) 处展开成幂级数，并写出收敛域。
✓ 标准解答

Step 1　因式分解并部分分式分解：

\[x^2-3x+2 = (x-1)(x-2)\]

设 \(\frac{1}{(x-1)(x-2)} = \frac{A}{x-1} + \frac{B}{x-2}\)，通分得 \(1 = A(x-2) + B(x-1)\)。

令 \(x=1\)：\(1 = A(-1) \Rightarrow A = -1\)。

令 \(x=2\)：\(1 = B(1) \Rightarrow B = 1\)。

于是

\[f(x) = \frac{1}{x-2} - \frac{1}{x-1} = \frac{1}{1-x} - \frac{1}{2-x}\]

Step 2　分别展开两项：

\[\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n,\qquad |x| < 1\]
\[\frac{1}{2-x} = \frac{1}{2}\cdot\frac{1}{1-\frac{x}{2}} = \frac{1}{2}\sum_{n=0}^{\infty}\left(\frac{x}{2}\right)^n = \sum_{n=0}^{\infty}\frac{x^n}{2^{n+1}},\qquad |x| < 2\]

Step 3　合并：

\[f(x) = \sum_{n=0}^{\infty} x^n - \sum_{n=0}^{\infty}\frac{x^n}{2^{n+1}} = \sum_{n=0}^{\infty}\left(1 - \frac{1}{2^{n+1}}\right)x^n = \sum_{n=0}^{\infty}\frac{2^{n+1}-1}{2^{n+1}}x^n\]

Step 4　确定收敛域。两个展开式分别要求 \(|x|<1\) 和 \(|x|<2\)，取交集得 \(|x|<1\)。

端点 \(x = \pm 1\) 时通项 \(\left(1-\frac{1}{2^{n+1}}\right)(\pm 1)^n\) 的绝对值趋于 1，不趋于 0，均发散。

答案：\(\boxed{\displaystyle\frac{1}{x^2-3x+2} = \sum_{n=0}^{\infty}\frac{2^{n+1}-1}{2^{n+1}}x^n,\quad x\in(-1,\;1)}\)

展开式的收敛域

如果展开式由多个级数相加得到，收敛域必须取各部分的交集。常见错误是只看最宽的限制（如 \(|x|<2\)），而忽略了另一部分要求 \(|x|<1\)。

⚠️ 常见错误

错误 1：端点忘记判断

收敛域必须单独代入端点验证敛散性。比值法/根值法只能给出开区间 \((-R,R)\)，端点处的敛散性必须用数项级数方法另行判断。这是最常见的丢分点。

错误 2：和函数漏写收敛域

求和函数题，最后一定要写出"该和函数在某某区间上成立"。和函数在收敛域外是没有定义的。

错误 3：展开后没写收敛域

展开题和收敛域题一样，必须注明成立范围。只写展开式不写范围，等于只完成了一半。

错误 4：求和函数时定常数出错

先导后积还原时，\(S(x) = S(0) + \int_0^x S'(t)\,dt\)。定 \(S(0)\) 必须回到原级数看常数项，不能把 \(x=0\) 直接代入求导后的式子。例如 \(\sum\frac{x^n}{n}\) 的 \(S(0)=0\)，但求导后 \(\frac{1}{1-x}\) 在 \(x=0\) 时为 1。

错误 5：缺项幂级数直接用标准比值法

对于缺项级数（如只含偶次项 \(\sum a_n x^{2n}\)），不能直接对 \(a_n\) 用 \(R = |a_n/a_{n+1}|\)。正确做法是令 \(t = x^2\) 化为标准形，或者用根值法 \(R = 1/\limsup\sqrt[2n]{|a_n|}\)。
