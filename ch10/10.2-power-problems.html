10.2 · 题型

幂级数 三类题型标准步骤

考试考三件事：收敛域、和函数、展开。步骤最固定，练熟就能拿满。

本页导航

  • 题型一：收敛域
  • 题型二：和函数
  • 题型三：函数展开
  • 常见错误

题型一：求收敛域

1

求收敛半径

\(R = \lim_{n\to\infty} \left|\frac{a_n}{a_{n+1}}\right|\)

2

写出收敛区间

\((-R, R)\)

3

单独判断端点

代入 \(x=R\) 和 \(x=-R\)，用数项级数方法判敛散。

4

写出收敛域

四种可能：\((-R,R), [-R,R), (-R,R], [-R,R]\)

例题收敛域
求 \(\sum_{n=1}^{\infty} \frac{x^n}{n}\) 的收敛域。
✓ 标准解答

\(a_n = \frac{1}{n}\)，\(R = \lim \frac{1/n}{1/(n+1)} = \lim \frac{n+1}{n} = 1\)

\(x=1\)：\(\sum \frac{1}{n}\) 调和级数，发散。

\(x=-1\)：\(\sum \frac{(-1)^n}{n}\) 交错级数，收敛。

收敛域为 \(\mathbf{[-1, 1)}\)

题型二：求和函数

核心口诀

"分母有 n → 求导消掉；分子有 n → 积分消掉"。目标永远是把级数变成等比级数 \(\sum x^n = \frac{1}{1-x}\)。

1

设 \(S(x)\)，确定收敛域

2

看 n 的位置，选择求导或积分

3

化简为等比级数，求和

4

逆操作还原（求导后积分，积分后求导）

5

代入特殊点（如 \(x=0\)）定常数 \(C\)

例题和函数
求 \(\sum_{n=1}^{\infty} n x^{n-1}\) 的和函数（\(|x|<1\)）。
✓ 标准解答

注意到 \(\int_0^x nt^{n-1}dt = x^n\)，所以先积分再求和：

\[\int_0^x S(t)dt = \sum_{n=1}^{\infty} x^n = \frac{x}{1-x}\]

两边求导：

\[S(x) = \left(\frac{x}{1-x}\right)' = \frac{1}{(1-x)^2}\]

题型三：函数展开成幂级数

1

识别"招牌"：变形匹配5个基本展开式

2

代入公式，写出展开式

3

注明收敛域

例题展开
将 \(f(x) = \frac{1}{2+x}\) 在 \(x=0\) 处展开成幂级数。
✓ 标准解答

变形：\(\frac{1}{2+x} = \frac{1}{2}\cdot\frac{1}{1+\frac{x}{2}} = \frac{1}{2}\cdot\frac{1}{1-(-\frac{x}{2})}\)

利用 \(\frac{1}{1-t} = \sum t^n\)（\(|t|<1\)），令 \(t = -\frac{x}{2}\)：

\[\frac{1}{2+x} = \frac{1}{2}\sum_{n=0}^{\infty}\left(-\frac{x}{2}\right)^n = \sum_{n=0}^{\infty}\frac{(-1)^n}{2^{n+1}}x^n\] 收敛域：\(\left|-\frac{x}{2}\right| < 1\)，即 \(|x| < 2\)。端点需单独验证。

⚠️ 常见错误

端点忘记判断

收敛域必须单独代入端点验证。这是最常见的丢分点。

和函数忘记写收敛域

求和函数题，最后一定要写出"该和函数在 xx 上成立"。

展开后没写收敛域

展开题和收敛域题一样，必须注明成立范围。
