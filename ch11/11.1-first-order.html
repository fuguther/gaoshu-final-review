一阶微分方程 识别 + 套公式

一句话：一阶方程的核心是3秒内识别类型，然后套用对应解法。
考试考四类：可分离、齐次、线性、伯努利。

🔍 四类型识别表（3秒定类型）

类型标准形式识别信号解法
可分离 \(\frac{dy}{dx} = f(x)g(y)\) 右边能拆成 x 的函数 × y 的函数 分离变量，两边积分
齐次 \(\frac{dy}{dx} = F\left(\frac{y}{x}\right)\) 右边是 \(\frac{y}{x}\) 的函数 令 \(u = \frac{y}{x}\)
线性 \(\frac{dy}{dx} + P(x)y = Q(x)\) y 和 y' 都是一次（无 y²、√y 等） 套通解公式
伯努利 \(\frac{dy}{dx} + P(x)y = Q(x)y^n\) 像线性但右边多了 \(y^n\) 令 \(z = y^{1-n}\)
优先级

如果方程同时满足多个类型，按这个顺序判断：可分离 → 齐次 → 线性 → 伯努利。可分离是最基础的"母型"，很多方程本质上都是可分离的变形。

📐 类型一：可分离变量

标准形式

\[\frac{dy}{dx} = f(x) \cdot g(y)\]

解法

1

分离变量

\(\frac{dy}{g(y)} = f(x)\,dx\)（假设 \(g(y) \neq 0\)）

2

两边积分

\(\int \frac{dy}{g(y)} = \int f(x)\,dx\)

3

解出 y

整理得到通解（含一个任意常数 \(C\)）。

例题 可分离变量
求 \( \frac{dy}{dx} = xy \) 的通解。
✓ 标准解答

Step 1　分离：\(\frac{dy}{y} = x\,dx\)（\(y \neq 0\)）

Step 2　积分：\(\ln|y| = \frac{x^2}{2} + C_1\)

Step 3　整理：\(|y| = e^{C_1}e^{x^2/2}\)

\[y = Ce^{x^2/2}\]
其中 \(C = \pm e^{C_1}\) 为任意常数（包含 \(C=0\) 对应 \(y=0\)）

📐 类型二：齐次方程

标准形式

\[\frac{dy}{dx} = F\left(\frac{y}{x}\right)\]

解法

1

换元

令 \(u = \frac{y}{x}\)，即 \(y = ux\)。则 \(\frac{dy}{dx} = u + x\frac{du}{dx}\)。

2

代入化简

原方程变为 \(u + x\frac{du}{dx} = F(u)\)，整理得 \(x\frac{du}{dx} = F(u) - u\)。

3

分离变量

\(\frac{du}{F(u)-u} = \frac{dx}{x}\)，化为可分离变量方程。

4

积分并回代

两边积分后用 \(u = \frac{y}{x}\) 回代。

识别技巧

如果方程右边出现 \(x+y\)、\(x-y\)、\(x^2+y^2\)、\(\frac{y}{x}\) 等结构，大概率是齐次方程。特别地，若分子分母各项次数相同（如 \(\frac{x^2+xy}{y^2}\)），也是齐次。

📐 类型三：一阶线性方程（最重要）

标准形式

\[\frac{dy}{dx} + P(x)y = Q(x)\]

通解公式（必须背熟）

\[y = e^{-\int P(x)\,dx}\left(\int Q(x)e^{\int P(x)\,dx}dx + C\right)\]
考场速记

记 \(I = \int P(x)\,dx\)，则公式简化为：

\[y = e^{-I}\left(\int Q(x)e^{I}dx + C\right)\]

口诀：先算积分 \(I\)，再算 \(e^I\)，然后乘 \(Q\) 再积分，最后乘 \(e^{-I}\)。

例题 一阶线性
求 \( \frac{dy}{dx} + 2xy = x \) 的通解。
✓ 标准解答

Step 1　识别：\(P(x)=2x, Q(x)=x\)

Step 2　\(I = \int 2x\,dx = x^2\)

Step 3　\(e^I = e^{x^2}\)

Step 4　\(\int Q e^I dx = \int x e^{x^2} dx = \frac{1}{2}e^{x^2}\)

Step 5　代入公式：

\[y = e^{-x^2}\left(\frac{1}{2}e^{x^2} + C\right) = \frac{1}{2} + Ce^{-x^2}\]

📐 类型四：伯努利方程

标准形式

\[\frac{dy}{dx} + P(x)y = Q(x)y^n \quad (n \neq 0, 1)\]

解法

1

换元

令 \(z = y^{1-n}\)，则 \(\frac{dz}{dx} = (1-n)y^{-n}\frac{dy}{dx}\)。

2

化为一阶线性

原方程两边乘 \((1-n)y^{-n}\)，得到关于 \(z\) 的一阶线性方程。

3

套用线性公式

解出 \(z\) 后，用 \(z = y^{1-n}\) 回代得 \(y\)。

📋 解法对照速查表

类型第一步核心操作结果形式
可分离拆成 f(x)g(y)分离 → 积分隐式或显式通解
齐次令 u = y/x换元 → 分离 → 回代含 C 的通解
线性确定 P(x), Q(x)套公式\(y = e^{-I}(\int Qe^I dx + C)\)
伯努利令 z = y^(1-n)化线性 → 套公式 → 回代含 C 的通解
