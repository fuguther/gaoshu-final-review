11.1 · 一阶 ODE（高数C必考 Q3）

一阶微分方程 习题全解

《高等数学（生化医农类）下册》习题 11.1 各型一阶方程求通解 / 特解 · 考场标准步骤

高数C 期末 Q3（30 分）必考微分方程求解。本页收齐习题 11.1 各型一阶方程——可分离、齐次、一阶线性、常数变易、伯努利、全微分——求通解（问题1）与初值特解（问题2）共 54 小题，A 路双解、对照书末官方答案抽查（ODE 通解等价形式较多，个别答案以等价形式呈现）。应用题（求曲线/温度/浓度等）不单独考，未收录。

1.求通解

(1)\(\displaystyle \frac{dy}{dx}=\frac{1+y^{2}}{(1+x^{2})xy}\)
方程改写为 \(\frac{y}{1+y^2}dy = \frac{1}{x(1+x^2)}dx\)
对右边分解 \(\frac{1}{x(1+x^2)} = \frac{1}{x} - \frac{x}{1+x^2}\)
两边积分得 \(\frac{1}{2}\ln(1+y^2) = \ln|x| - \frac{1}{2}\ln(1+x^2) + \frac{1}{2}\ln C\)
整理为 \(\ln(1+y^2) = 2\ln|x| - \ln(1+x^2) + \ln C = \ln\frac{Cx^2}{1+x^2}\)
答案\(1+y^2 = \frac{Cx^2}{1+x^2}\)，即 \((1+x^2)(1+y^2) = Cx^2\)（\(C>0\) 为任意常数），或等价写成 \(y^2 = \frac{(C-1)x^2-1}{1+x^2}\)
(2)\(\displaystyle a\left(x\frac{dy}{dx}+2y\right)=xy\frac{dy}{dx}\)
方程整理为 \(a x\frac{dy}{dx} + 2ay = xy\frac{dy}{dx}\)，即 \((ax-xy)\frac{dy}{dx} = -2ay\)，分离变量得 \(\frac{y-a}{y}dy = \frac{2a}{x}dx\)
左边拆分为 \(\left(1-\frac{a}{y}\right)dy = \frac{2a}{x}dx\)
积分得 \(y - a\ln|y| = 2a\ln|x| + C_1\)
整理为 \(y = a\ln|y| + a\ln x^2 + C_1 = a\ln|yx^2| + C_1\)
指数化得 \(e^{y/a} = e^{C_1/a}|yx^2|\)，即 \(x^2 y = C e^{y/a}\)（\(C = \pm e^{C_1/a}\)）
答案通解为 \(x^2 y = C e^{y/a}\)（\(C\) 为任意常数），或等价写成 \(y = a\ln|x^2 y| + C\)
(3)\(\displaystyle y-xy'=a(y^{2}+y')\)
识别方程为 \( y - xy' = a(y^2 + y') \)，整理为关于 \( y' \) 的显式形式
\[分离变量\]
对右边分式分解 \( \frac{1}{y(1-ay)} = \frac{1}{y} + \frac{a}{1-ay} \)，积分得
\[整理为\]
取指数并令 \( C = e^{C_1} \)，化简得隐式通解
答案通解为 \( \frac{y}{1-ay} = C(x+a)^a e^x \)（\( C \) 为任意常数），或显式写为 \( y = \frac{C(x+a)^a e^x}{1 + aC(x+a)^a e^x} \)
(4)\(\displaystyle \frac{dy}{dx}=10^{x+y}\)
识别为可分离变量方程：\(\frac{dy}{dx} = 10^x \cdot 10^y\)
分离变量：\(10^{-y}dy = 10^x dx\)
积分两边：\(\int 10^{-y}dy = \int 10^x dx\)
计算积分：\(\frac{10^{-y}}{-\ln 10} = \frac{10^x}{\ln 10} + C_1\)
整理得：\(-10^{-y} = 10^x + C\)（其中 \(C = C_1\ln 10\)）
即 \(10^x + 10^{-y} = C\)，或写成 \(10^{-y} = C - 10^x\)
答案\(y = -\log_{10}(C - 10^x)\)，或等价形式 \(10^x + 10^{-y} = C\)（\(C\) 为任意常数）
(5)\(\displaystyle \sqrt{1+x^{2}}dy-\sqrt{1-y^{2}}dx=0\)
识别为可分离变量方程，\[\sqrt{1+x^{2}}\,dy = \sqrt{1-y^{2}}\,dx\]
分离变量得 \[\frac{dy}{\sqrt{1-y^{2}}} = \frac{dx}{\sqrt{1+x^{2}}}\]
两边积分，\[\arcsin y = \ln\left(x+\sqrt{1+x^{2}}\right)+C\]
答案通解为 \[y = \sin\left[\ln\left(x+\sqrt{1+x^{2}}\right)+C\right]\] 或等价地 \[\arcsin y = \text{arsinh}\,x + C\]
(6)\(\displaystyle d\rho+\rho\tan\theta d\theta=0\)
识别方程：\(d\rho + \rho\tan\theta\, d\theta = 0\)，即 \(\frac{d\rho}{\rho} = -\tan\theta\, d\theta\)，为可分离变量方程
积分：\[\int\frac{d\rho}{\rho} = -\int\tan\theta\, d\theta = \int\frac{d\cos\theta}{\cos\theta}\]
得 \(\ln|\rho| = \ln|\cos\theta| + C_1\)，即 \(\rho = C\cos\theta\)（\(C = \pm e^{C_1}\)，含 \(\rho=0\)）
答案\(\rho = C\cos\theta\)（\(C\) 为任意常数）
(7)\(\displaystyle (1-x)dy-y^{2}dx=0\)
识别方程为可分离变量型：\((1-x)dy = y^{2}dx\)
分离变量（\(y \neq 0\)）：\(\frac{dy}{y^{2}} = \frac{dx}{1-x}\)
两边积分：\(\int y^{-2}dy = \int \frac{dx}{1-x}\)
计算得：\(-\frac{1}{y} = -\ln|1-x| + C_{1}\)
整理并令 \(C = -C_{1}\)：\(\frac{1}{y} = \ln|1-x| + C\)
显式通解：\(y = \frac{1}{\ln|1-x| + C}\)
检验特解：\(y = 0\) 代入原方程满足，但分离变量时已排除，故为奇解
答案通解为 \(y = \frac{1}{\ln|1-x| + C}\)（\(C\) 为任意常数），另有特解 \(y = 0\)
(8)\(\displaystyle (x+2y)dx+(2x-3y)dy=0\)
识别为齐次方程，令 \( y = ux \)，则 \( dy = u\,dx + x\,du \)
代入得 \((x+2ux)\,dx + (2x-3ux)(u\,dx+x\,du) = 0\)，整理为 \(x(1+2u)\,dx + x(2-3u)(u\,dx+x\,du) = 0\)
展开并合并 \(dx\) 项：\([1+2u+2u-3u^2]\,dx + x(2-3u)\,du = 0\)，即 \((1+4u-3u^2)\,dx + x(2-3u)\,du = 0\)
分离变量：\(\frac{dx}{x} = \frac{3u-2}{1+4u-3u^2}\,du = \frac{3u-2}{(1-u)(1+3u)}\,du\)
对右边作部分分式：\(\frac{3u-2}{(1-u)(1+3u)} = \frac{A}{1-u} + \frac{B}{1+3u}\)，解得 \(A = -\frac{1}{4}\)，\(B = \frac{9}{4}\)，即 \(\frac{1}{4}\left(\frac{-1}{1-u} + \frac{9}{1+3u}\right)\)
积分：\(\ln|x| = \frac{1}{4}\left(\ln|1+3u| - \ln|1-u|\right) + C_1\)，即 \(4\ln|x| = \ln\left|\frac{1+3u}{1-u}\right| + C_2\)
回代 \(u = \frac{y}{x}\) 并整理：\(x^4 = C\cdot\frac{x+3y}{x-y}\)，即 \(x^4(x-y) = C(x+3y)\)，或写成 \(x^3(x-y) = C_3(x+3y)\) 检验后得 \(x^4\frac{x-y}{x+3y} = C\)
答案\(x^4 - x^3y = C(x+3y)\)，或等价地写成 \((x-y)(x+3y)^3 = C_1\) 经验证，标准形式为 \((x+3y)^3(x-y) = C\)
(9)\(\displaystyle (3x+5y)dx+(4x+6y)dy=0\)
识别为齐次方程，令 \(u=\frac{y}{x}\)，则 \(y=ux\)，\(dy=u\,dx+x\,du\)
代入得 \((3x+5ux)\,dx+(4x+6ux)(u\,dx+x\,du)=0\)，即 \(x(3+5u)\,dx+x(4+6u)(u\,dx+x\,du)=0\)
除以 \(x\) 并整理：\((3+5u+4u+6u^2)\,dx+x(4+6u)\,du=0\)，即 \((6u^2+9u+3)\,dx+x(4+6u)\,du=0\)
分离变量：\(\frac{dx}{x}=-\frac{4+6u}{6u^2+9u+3}\,du=-\frac{2(2+3u)}{3(2u^2+3u+1)}\,du=-\frac{2(2+3u)}{3(2u+1)(u+1)}\,du\)
对右边分式分解：\(\frac{2(2+3u)}{(2u+1)(u+1)}=\frac{A}{2u+1}+\frac{B}{u+1}\)，解得 \(A=2\)，\(B=2\)，故 \(\frac{dx}{x}=-\frac{2}{3}\left(\frac{1}{2u+1}+\frac{1}{u+1}\right)du\)
积分得 \(\ln|x|=-\frac{2}{3}\left(\frac{1}{2}\ln|2u+1|+\ln|u+1|\right)+C_1=-\frac{1}{3}\ln|2u+1|-\frac{2}{3}\ln|u+1|+C_1\)
整理：\(3\ln|x|+\ln|2u+1|+2\ln|u+1|=3C_1\)，即 \(\ln|x^3(2u+1)(u+1)^2|=3C_1\)
回代 \(u=\frac{y}{x}\)：\(x^3\left(2\frac{y}{x}+1\right)\left(\frac{y}{x}+1\right)^2=C\)，即 \(x^3\cdot\frac{2y+x}{x}\cdot\frac{(y+x)^2}{x^2}=C\)
答案\((x+2y)(x+y)^2=C\) 或展开为 \(x^3+4x^2y+5xy^2+2y^3=C\)
(10)\(\displaystyle y'=xy+x+y+1\)
识别方程结构：\( y' = xy + x + y + 1 = (x+1)(y+1) \)，为可分离变量型
令 \( u = y+1 \)，则 \( u' = y' \)，方程化为 \( u' = (x+1)u \)
分离变量：\( \frac{du}{u} = (x+1)\,dx \)（\( u \neq 0 \)）
积分得 \( \ln|u| = \frac{x^2}{2} + x + C_1 \)，即 \( |u| = e^{C_1}e^{\frac{x^2}{2}+x} \)
去掉绝对值并含 \( u=0 \) 情形：\( u = Ce^{\frac{x^2}{2}+x} \)（\( C \in \mathbb{R} \)）
回代 \( u = y+1 \)：\( y+1 = Ce^{\frac{x^2}{2}+x} \)
答案\( y = Ce^{\frac{x^2}{2}+x} - 1 \)，其中 \( C \) 为任意常数
(11)\(\displaystyle 2(x+y)dx+ydy=0\)
识别方程类型：\(2(x+y)\,dx + y\,dy = 0\)，即 \(\frac{dy}{dx} = -\frac{2(x+y)}{y}\)，属可化为齐次方程或需换元的一阶方程。令 \(u = x + y\)，则 \(y = u - x\)，\(\frac{dy}{dx} = \frac{du}{dx} - 1\)
代入得 \(\frac{du}{dx} - 1 = -\frac{2u}{u-x} = \frac{2u}{x-u}\)，整理为 \(\frac{du}{dx} = \frac{2u + x - u}{x - u} = \frac{u + x}{x - u} = \frac{u/x + 1}{1 - u/x}\)
令 \(v = \frac{u}{x}\)，则 \(u = vx\)，\(\frac{du}{dx} = v + x\frac{dv}{dx}\)，代入得 \(v + x\frac{dv}{dx} = \frac{v+1}{1-v}\)
分离变量：\(x\frac{dv}{dx} = \frac{v+1}{1-v} - v = \frac{v+1 - v(1-v)}{1-v} = \frac{v^2+1}{1-v}\)，即 \(\frac{1-v}{v^2+1}\,dv = \frac{dx}{x}\)
积分：\(\int\frac{1}{v^2+1}\,dv - \int\frac{v}{v^2+1}\,dv = \int\frac{dx}{x}\)，得 \(\arctan v - \frac{1}{2}\ln(v^2+1) = \ln|x| + C_1\)
回代 \(v = \frac{u}{x} = \frac{x+y}{x}\)：\(\arctan\frac{x+y}{x} - \frac{1}{2}\ln\left(\frac{(x+y)^2}{x^2}+1\right) = \ln|x| + C_1\)
化简：\(\arctan\frac{x+y}{x} - \frac{1}{2}\ln\frac{(x+y)^2+x^2}{x^2} = \ln|x| + C_1\)，即 \(\arctan\frac{x+y}{x} - \frac{1}{2}\ln\left((x+y)^2+x^2\right) + \ln|x| = \ln|x| + C_1\)
整理得 \(\arctan\frac{x+y}{x} = \frac{1}{2}\ln\left(2x^2+2xy+y^2\right) + C\)，或等价写成 \(2\arctan\frac{x+y}{x} = \ln\left(2x^2+2xy+y^2\right) + C_2\)
答案\(2\arctan\frac{x+y}{x} = \ln\left(2x^2+2xy+y^2\right) + C\)，或隐式通解 \(\arctan\left(1+\frac{y}{x}\right) = \frac{1}{2}\ln\left((x+y)^2+x^2\right) + C\)

(13)\(\displaystyle (x+4y)dx+2xdy=0\)
方程 \((x+4y)\,dx + 2x\,dy = 0\) 改写为 \(\frac{dy}{dx} = -\frac{x+4y}{2x} = -\frac{1}{2} - \frac{2y}{x}\)，识别为齐次/可化为齐次型，令 \(y = ux\)
代入 \(y = ux\), \(dy = u\,dx + x\,du\) 得 \(u + x\frac{du}{dx} = -\frac{1}{2} - 2u\)，即 \(x\frac{du}{dx} = -\frac{1}{2} - 3u\)
分离变量 \(\frac{du}{3u+\frac{1}{2}} = -\frac{dx}{x}\)，即 \(\frac{2\,du}{6u+1} = -\frac{dx}{x}\)
积分得 \(\frac{1}{3}\ln|6u+1| = -\ln|x| + C_1\)，即 \(\ln|6u+1| = -3\ln|x| + C_2\)
回代 \(u = \frac{y}{x}\) 得 \(6\frac{y}{x}+1 = \frac{C}{x^3}\)，即 \(\frac{6y+x}{x} = \frac{C}{x^3}\)
答案\[x^2(x+6y) = C\] 或等价写成 \((x+6y)x^2 = C\)（\(C\) 为任意常数）
(14)\(\displaystyle (2x^{2}+y^{2})dx+(2xy+3y^{2})dy=0\)
验证恰当性，\(\frac{\partial M}{\partial y} = 2y\)，\(\frac{\partial N}{\partial x} = 2y\)，方程为全微分方程
求势函数，\(u(x,y) = \int (2x^2+y^2)\,dx = \frac{2}{3}x^3 + xy^2 + \varphi(y)\)
由 \(\frac{\partial u}{\partial y} = 2xy + \varphi'(y) = 2xy + 3y^2\)，得 \(\varphi'(y) = 3y^2\)，\(\varphi(y) = y^3\)
通解为 \(\frac{2}{3}x^3 + xy^2 + y^3 = C\)，或等价写为 \(2x^3 + 3xy^2 + 3y^3 = C_1\)
答案\(2x^3 + 3xy^2 + 3y^3 = C\)
(15)\(\displaystyle (e^{x+y}-e^{x})dx+(e^{x+y}+e^{y})dy=0\)
识别方程为全微分方程，验证 \(\frac{\partial M}{\partial y} = e^{x+y}\) 与 \(\frac{\partial N}{\partial x} = e^{x+y}\) 相等，其中 \(M = e^{x+y}-e^x\)，\(N = e^{x+y}+e^y\)
重组微分形式：\(e^{x+y}(dx+dy) - e^x dx + e^y dy = 0\)，即 \(d(e^{x+y}) - d(e^x) + d(e^y) = 0\)
积分得 \(e^{x+y} - e^x + e^y = C\)
答案\(e^{x+y} - e^x + e^y = C\)（或等价写成 \((e^x+1)(e^y-1) = C_1\)）

代入原方程 \((x^2 - u^2x^2)(u\,dx + x\,du) - 2ux^2\,dx = 0\)，整理得

(18)\(\displaystyle (2x+3y)y'-y=0\)
方程改写为 \( \frac{dx}{dy} = \frac{2x+3y}{y} = \frac{2x}{y} + 3 \)，即 \( \frac{dx}{dy} - \frac{2}{y}x = 3 \)（关于 \( x \) 的一阶线性方程，\( y \) 为自变量）
积分因子 \( \mu = e^{-\int \frac{2}{y}dy} = e^{-2\ln|y|} = \frac{1}{y^2} \)
通解公式 \( x = y^2\left(\int \frac{3}{y^2}dy + C\right) = y^2\left(-\frac{3}{y} + C\right) = Cy^2 - 3y \)
答案\( x = Cy^2 - 3y \)（或 \( x + 3y = Cy^2 \)，\( C \) 为任意常数）
(19)\(\displaystyle x\frac{dy}{dx}-2y=2x\)
方程化为标准形式 \(\frac{dy}{dx} - \frac{2}{x}y = 2\)，识别为一阶线性非齐次方程
积分因子 \(\mu(x) = e^{\int -\frac{2}{x}dx} = e^{-2\ln|x|} = x^{-2}\)
乘以积分因子得 \(\frac{d}{dx}\left(x^{-2}y\right) = 2x^{-2}\)
积分得 \(x^{-2}y = \int 2x^{-2}dx = -2x^{-1} + C\)
解出 \(y = x^2\left(-\frac{2}{x} + C\right) = -2x + Cx^2\)
答案\(y = Cx^2 - 2x\)，其中 \(C\) 为任意常数
(20)\(\displaystyle \frac{dy}{dx}-y=-2e^{-x}\)
一阶线性方程标准形：\(\frac{dy}{dx} + P(x)y = Q(x)\)，此处 \(P(x) = -1\)，\(Q(x) = -2e^{-x}\)
积分因子 \(\mu(x) = e^{\int -1\,dx} = e^{-x}\)
方程两边乘 \(\mu(x)\)：\(e^{-x}\frac{dy}{dx} - e^{-x}y = -2e^{-2x}\)，即 \(\frac{d}{dx}\left(ye^{-x}\right) = -2e^{-2x}\)
积分得 \(ye^{-x} = \int -2e^{-2x}\,dx = e^{-2x} + C\)
通解 \(y = e^{x}\left(e^{-2x} + C\right) = e^{-x} + Ce^{x}\)
答案\(y = Ce^{x} + e^{-x}\)（\(C\) 为任意常数）
(21)\(\displaystyle \frac{ds}{dt}-s\cot t=1-(t+2)\cot t\)
识别一阶线性方程标准形 \(\frac{ds}{dt} - s\cot t = 1-(t+2)\cot t\)，其中 \(P(t)=-\cot t\)
求积分因子 \(\mu(t)=e^{\int -\cot t\,dt}=e^{-\ln|\sin t|}=\frac{1}{\sin t}\)（取 \(t\in(0,\pi)\) 时 \(\sin t>0\)）
方程两边乘积分因子得 \(\frac{d}{dt}\left(\frac{s}{\sin t}\right)=\frac{1}{\sin t}-\frac{(t+2)\cos t}{\sin^2 t}\)
右边积分：\(\int\frac{dt}{\sin t}-\int\frac{(t+2)\cos t}{\sin^2 t}dt\)；第一项 \(=\ln\left|\tan\frac{t}{2}\right|\)，第二项用分部积分 \(\int(t+2)d\left(-\frac{1}{\sin t}\right)=-\frac{t+2}{\sin t}+\int\frac{dt}{\sin t}\)
合并得 \(\frac{s}{\sin t}=-\frac{t+2}{\sin t}+C\)，即 \(s=-(t+2)+C\sin t\)

计算 \( \frac{1}{N}\left(\frac{\partial M}{\partial y} - \frac{\partial N}{\partial x}\right) = \frac{1}{y^3 - \ln x} \cdot \frac{2}{x} \) 仅依赖于 \( y \) 不成立；改求 \( \frac{1}{M}\left(\frac{\partial N}{\partial x} - \frac{\partial M}{\partial y}\right) = \frac{x}{y} \cdot \left(-\frac{2}{x}\right) = -\frac{2}{y} \) 仅依赖于 \( y \)
取积分因子 \( \mu(y) = e^{\int -\frac{2}{y}dy} = e^{-2\ln|y|} = \frac{1}{y^2} \)（设 \( y > 0 \)）
乘以积分因子得恰当方程 \( \frac{1}{xy}dx + \left(y - \frac{\ln x}{y^2}\right)dy = 0 \)，验证 \( \frac{\partial}{\partial y}\left(\frac{1}{xy}\right) = -\frac{1}{xy^2} = \frac{\partial}{\partial x}\left(y - \frac{\ln x}{y^2}\right) \)
求势函数 \( u(x,y) \)，由 \( \frac{\partial u}{\partial x} = \frac{1}{xy} \) 积分得 \( u = \frac{\ln x}{y} + \varphi(y) \)；由 \( \frac{\partial u}{\partial y} = -\frac{\ln x}{y^2} + \varphi'(y) = y - \frac{\ln x}{y^2} \) 得 \( \varphi'(y) = y \)，故 \( \varphi(y) = \frac{y^2}{2} \)
通解为 \( \frac{\ln x}{y} + \frac{y^2}{2} = C \)，即 \( 2\ln x + y^3 = 2Cy \) 或等价写为 \( y^3 + 2\ln|x| = Cy \)（常数 \( C \) 重新标记）
答案\( \frac{\ln|x|}{y} + \frac{y^2}{2} = C \) 或等价形式 \( 2\ln|x| + y^3 = Cy \)（\( C \) 为任意常数），特解 \( y = 0 \)
(44)\(\displaystyle (x\cos y-y\sin y)dy+(x\sin y+y\cos y)dx=0\)
识别方程为恰当方程或寻找积分因子。设 \(M=x\sin y+y\cos y\), \(N=x\cos y-y\sin y\)，计算偏导数：
计算 \(\frac{1}{N}\left(\frac{\partial M}{\partial y}-\frac{\partial N}{\partial x}\right)=\frac{x\cos y+\cos y-y\sin y-\cos y}{x\cos y-y\sin y}=\frac{x\cos y-y\sin y}{x\cos y-y\sin y}=1\)，仅为 \(x\) 的函数（实为常数），故积分因子 \(\mu=e^{\int 1\,dx}=e^x\)。
以 \(\mu=e^x\) 乘原方程，新方程 \(M_1=e^x(x\sin y+y\cos y)\), \(N_1=e^x(x\cos y-y\sin y)\) 为恰当方程。求势函数 \(u(x,y)\)：

2.求满足初始条件的特解

(1)\(\displaystyle \frac{dx}{y}+\frac{4dy}{x}=0,\ 当\ x=4\ 时,\ y=2\)
方程 \(\frac{dx}{y}+\frac{4dy}{x}=0\) 为可分离变量方程，整理得 \(x\,dx + 4y\,dy = 0\)
积分得 \(\int x\,dx + \int 4y\,dy = C\)，即 \(\frac{x^2}{2} + 2y^2 = C\)
由初始条件 \(x=4,\ y=2\) 代入：\(\frac{16}{2} + 2\cdot 4 = 8 + 8 = 16 = C\)
故通解为 \(\frac{x^2}{2} + 2y^2 = 16\)，即 \(x^2 + 4y^2 = 32\)
答案\(x^2 + 4y^2 = 32\)（或写成 \(\frac{x^2}{32} + \frac{y^2}{8} = 1\)）
(2)\(\displaystyle (x^{2}+y^{2})dx=2xydy,\ 当\ x=1\ 时,\ y=0\)
方程化为 \(\frac{dy}{dx} = \frac{x^2+y^2}{2xy}\)，识别为齐次方程，令 \(y = ux\)，则 \(\frac{dy}{dx} = u + x\frac{du}{dx}\)
代入得 \(u + x\frac{du}{dx} = \frac{1+u^2}{2u}\)，整理得 \(x\frac{du}{dx} = \frac{1-u^2}{2u}\)
分离变量 \(\frac{2u}{1-u^2}du = \frac{dx}{x}\)，积分得 \(-\ln|1-u^2| = \ln|x| + C_1\)
化简得 \(x(1-u^2) = C\)，回代 \(u = \frac{y}{x}\) 得 \(x - \frac{y^2}{x} = C\)，即 \(x^2 - y^2 = Cx\)
由初始条件 \(x=1, y=0\) 得 \(C = 1\)，故 \(x^2 - y^2 = x\)
解出 \(y^2 = x^2 - x = x(x-1)\)，由 \(x=1\) 时 \(y=0\) 及连续性，取 \(y = \sqrt{x^2-x}\)（需 \(x \geq 1\) 或 \(x \leq 0\)，结合初始点邻域取 \(x \geq 1\)）
答案\(y^2 = x^2 - x\)，或写成 \(y = \sqrt{x^2-x}\)（\(x \geq 1\)）
(3)\(\displaystyle xdy-ydx=\sqrt{x^{2}+y^{2}}dx,\ 当\ x=\frac{1}{2}\ 时,\ y=0\)
方程改写为 \(x\,dy = \left(y + \sqrt{x^2+y^2}\right)dx\)，即 \(\frac{dy}{dx} = \frac{y + \sqrt{x^2+y^2}}{x}\)，此为齐次方程。
令 \(y = ux\)，则 \(\frac{dy}{dx} = u + x\frac{du}{dx}\)，代入得 \(u + x\frac{du}{dx} = u + \sqrt{1+u^2}\)，即 \(x\frac{du}{dx} = \sqrt{1+u^2}\)。
分离变量：\(\frac{du}{\sqrt{1+u^2}} = \frac{dx}{x}\)，积分得 \(\ln\left(u + \sqrt{1+u^2}\right) = \ln|x| + C\)。
由初始条件 \(x=\frac{1}{2}, y=0\) 得 \(u=0\)，代入：\(\ln(1) = \ln\frac{1}{2} + C\)，故 \(C = \ln 2\)。
回代 \(u = \frac{y}{x}\)：\(\frac{y}{x} + \sqrt{1+\frac{y^2}{x^2}} = 2x\)，即 \(y + \sqrt{x^2+y^2} = 2x^2\)。
由 \(\sqrt{x^2+y^2} = 2x^2 - y\) 平方得 \(x^2+y^2 = 4x^4 - 4x^2y + y^2\)，整理得 \(x^2(1-4x^2+4y) = 0\)，即 \(y = x^2 - \frac{1}{4}\)。
答案\(y = x^2 - \frac{1}{4}\)（或等价写成 \(y = x^2 - \frac{1}{4}\)，验证满足原方程及初始条件）
(4)\(\displaystyle \sin y\cdot\cos xdy=\cos y\cdot\sin xdx,\ 当\ x=0\ 时,\ y=\frac{\pi}{4}\)
方程化为可分离变量形式：\[\frac{\sin y}{\cos y}dy=\frac{\sin x}{\cos x}dx\]，即 \[\tan y\,dy=\tan x\,dx\]
积分得：\[-\ln|\cos y|=-\ln|\cos x|+C_1\]，即 \[\ln|\cos y|=\ln|\cos x|+C_2\]
指数化简得：\[\cos y=C\cos x\]（其中 \(C=\pm e^{C_2}\)）
代入初始条件 \(x=0,\ y=\frac{\pi}{4}\)：\[\cos\frac{\pi}{4}=C\cos 0\Rightarrow\frac{\sqrt{2}}{2}=C\]
故特解为：\[\cos y=\frac{\sqrt{2}}{2}\cos x\]，即 \[y=\arccos\left(\frac{\sqrt{2}}{2}\cos x\right)\]
答案\[y=\arccos\left(\frac{\sqrt{2}}{2}\cos x\right)\]
(5)\(\displaystyle \frac{dy}{dx}-\frac{2y}{x}=x^{2}e^{x},\ 当\ x=1\ 时,\ y=0\)
识别为一阶线性方程，标准形式 \(\frac{dy}{dx} + P(x)y = Q(x)\)，其中 \(P(x) = -\frac{2}{x}\)，\(Q(x) = x^2 e^x\)
积分因子 \(\mu(x) = e^{\int -\frac{2}{x}dx} = e^{-2\ln|x|} = x^{-2}\)
方程两边乘积分因子得 \(\frac{d}{dx}\left(x^{-2}y\right) = e^x\)
积分得 \(x^{-2}y = e^x + C\)，即 \(y = x^2(e^x + C)\)
代入初始条件 \(x=1, y=0\)：\(0 = 1\cdot(e + C)\)，得 \(C = -e\)
答案\(y = x^2(e^x - e)\)
(6)\(\displaystyle \frac{dy}{dx}+\frac{2y}{x}=\frac{1}{x^{2}},\ 当\ x=1\ 时,\ y=2\)
识别为一阶线性方程，标准形式 \(\frac{dy}{dx}+P(x)y=Q(x)\)，其中 \(P(x)=\frac{2}{x}\)，\(Q(x)=\frac{1}{x^2}\)
积分因子 \(\mu(x)=e^{\int\frac{2}{x}dx}=e^{2\ln|x|}=x^2\)（取 \(x>0\)）
方程化为 \(\frac{d}{dx}(x^2y)=x^2\cdot\frac{1}{x^2}=1\)
积分得 \(x^2y=\int 1\,dx=x+C\)，即 \(y=\frac{x+C}{x^2}=\frac{1}{x}+\frac{C}{x^2}\)
代入初始条件 \(x=1,y=2\)：\(2=1+C\)，得 \(C=1\)
答案\(y=\frac{1}{x}+\frac{1}{x^2}=\frac{x+1}{x^2}\)
(7)\(\displaystyle \frac{dy}{dx}+y\tan x=\sec x,\ 当\ x=0\ 时,\ y=-1\)
识别为一阶线性方程，标准形式 \(\frac{dy}{dx}+P(x)y=Q(x)\)，其中 \(P(x)=\tan x\)，\(Q(x)=\sec x\)
积分因子 \(\mu(x)=e^{\int\tan x\,dx}=e^{-\ln|\cos x|}=\sec x\)（取 \(x=0\) 附近 \(\cos x>0\) 的分支）
方程两边乘 \(\sec x\) 得 \(\frac{d}{dx}(y\sec x)=\sec^2 x\)
积分得 \(y\sec x=\tan x+C\)，即 \(y=\sin x+C\cos x\)
代入 \(y(0)=-1\)：\(-1=0+C\cdot 1\)，故 \(C=-1\)
答案\(y=\sin x-\cos x\)
(8)\(\displaystyle \frac{dy}{dx}-\frac{2y}{x+1}=(x+1)^{3},\ 当\ x=0\ 时,\ y=1\)
识别为一阶线性方程，标准形式 \(\frac{dy}{dx}+P(x)y=Q(x)\)，其中 \(P(x)=-\frac{2}{x+1}\)，\(Q(x)=(x+1)^3\)
积分因子 \(\mu(x)=e^{\int -\frac{2}{x+1}dx}=e^{-2\ln|x+1|}=(x+1)^{-2}\)
方程两边乘 \(\mu(x)\) 得 \(\frac{d}{dx}\left[y(x+1)^{-2}\right]=(x+1)\)
积分得 \(y(x+1)^{-2}=\frac{(x+1)^2}{2}+C\)，即 \(y=\frac{(x+1)^4}{2}+C(x+1)^2\)
代入 \(x=0,y=1\)：\(1=\frac{1}{2}+C\)，得 \(C=\frac{1}{2}\)
答案\(y=\frac{(x+1)^4+(x+1)^2}{2}\)
(9)\(\displaystyle x(1+y^{2})dx-y(1+x^{2})dy=0,\ 当\ x=0\ 时,\ y=2\)
方程为可分离变量型：\(\frac{x}{1+x^{2}}dx = \frac{y}{1+y^{2}}dy\)
积分得：\(\frac{1}{2}\ln(1+x^{2}) = \frac{1}{2}\ln(1+y^{2}) + C_{1}\)，即 \(\ln(1+x^{2}) = \ln(1+y^{2}) + 2C_{1}\)
去对数：\(1+x^{2} = C(1+y^{2})\)，其中 \(C = e^{2C_{1}} > 0\)
代入初始条件 \(x=0, y=2\)：\(1 = C(1+4) = 5C\)，故 \(C = \frac{1}{5}\)
特解为：\(1+x^{2} = \frac{1}{5}(1+y^{2})\)，整理得 \(5(1+x^{2}) = 1+y^{2}\)，即 \(y^{2} = 5x^{2}+4\)
答案\(y = \sqrt{5x^{2}+4}\)（取正号因 \(y(0)=2>0\)）
(10)\(\displaystyle y(x)=\int_{0}^{x}y(t)dt+x+1\)
对方程 \(y(x)=\int_{0}^{x}y(t)dt+x+1\) 两边关于 \(x\) 求导，由微积分基本定理得 \(y'(x)=y(x)+1\)
识别为一阶线性微分方程（亦是可分离变量方程），分离变量：\(\frac{dy}{y+1}=dx\)
积分得 \(\ln|y+1|=x+C_1\)，即 \(y+1=Ce^{x}\)，通解为 \(y=Ce^{x}-1\)
由原方程令 \(x=0\) 求初始条件：\(y(0)=\int_{0}^{0}y(t)dt+0+1=1\)
代入通解：\(1=Ce^{0}-1\)，得 \(C=2\)
答案\(y=2e^{x}-1\)
