11.2 · 题型

二阶线性 标准步骤与例题

特征方程 + 待定系数法。

题型一：齐次方程

例题齐次
求 \(y'' - 3y' + 2y = 0\) 的通解。
✓ 标准解答

特征方程：\(r^2 - 3r + 2 = 0\)，根 \(r_1=1, r_2=2\)

\(y = C_1e^x + C_2e^{2x}\)

题型二：非齐次方程（共振）

例题共振
求 \(y'' - 3y' + 2y = e^x\) 的通解。
✓ 标准解答

齐次通解：\(Y = C_1e^x + C_2e^{2x}\)

\(\lambda=1\) 是特征根（单根），设 \(y^* = Axe^x\)

代入求得 \(A = -1\)

\(y = C_1e^x + C_2e^{2x} - xe^x\)
常见错误

共振时忘记乘 \(x\) 是最大丢分点。如果强迫项的指数和特征根撞型，特解必须乘 \(x\)。
