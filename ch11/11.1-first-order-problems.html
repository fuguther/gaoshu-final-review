11.1 · 题型

一阶方程 标准步骤与例题

拿到一阶微分方程，3 秒判类型：能分离就分离，不能分离看齐次，
齐次不是看线性，线性不是看伯努利。判完类型套标准步骤，不丢分。

🌲 四步决策树

Step 1：看能不能写成 \(g(y)\,dy = f(x)\,dx\)
├─ 能 → 可分离变量
└─ 不能 → 继续 Step 2
Step 2：看右端能不能写成 \(f(y/x)\) 的形式
├─ 能 → 齐次方程（令 \(u = y/x\)）
└─ 不能 → 继续 Step 3
Step 3：看能不能写成 \(y'+P(x)y=Q(x)\)
├─ 能，且 \(y\) 的幂次只有 1 → 一阶线性（积分因子法）
└─ 右端还有 \(y^n\)（\(n\neq 0,1\)）→ 伯努利方程
考场口诀

"分离齐次线伯努，一步一步往下走；代换回代别漏掉，线性公式符号要记牢。"

题型一：可分离变量方程

标准步骤

1

移项，把含 \(y\) 的放左边，含 \(x\) 的放右边

\(\displaystyle g(y)\,dy = f(x)\,dx\)。如果分离时两边同除了一个可能为 0 的因子，注意检验是否漏掉特解。

2

两边同时积分

\(\displaystyle\int g(y)\,dy = \int f(x)\,dx\)，分别求不定积分。

3

解出 \(y\)（或隐式通解）

如果能显式解出 \(y = \varphi(x,C)\) 最好；解不出就保留隐式形式。注意只写一个积分常数。

例题 1 可分离变量 · 基本型
求微分方程 \(\displaystyle\frac{dy}{dx} = xy\) 的通解。
✓ 标准解答

Step 1　当 \(y\neq 0\) 时，分离变量：

\[\frac{dy}{y} = x\,dx\]

Step 2　两边积分：

\[\int\frac{dy}{y} = \int x\,dx \quad\Rightarrow\quad \ln|y| = \frac{x^2}{2} + C_1\]

Step 3　去对数，令 \(C = \pm e^{C_1}\)（可正可负），则

\[y = Ce^{\frac{x^2}{2}}\]

检验 \(y=0\)：代入原方程，左边 \(y'=0\)，右边 \(xy=0\)，满足。但 \(y=0\) 已包含在 \(C=0\) 的情形中，故无需额外写出。

答案：\(\boxed{y = Ce^{\frac{x^2}{2}}}\)

验证：\(y' = Cx e^{\frac{x^2}{2}} = x\cdot Ce^{\frac{x^2}{2}} = xy\) ✓

例题 2 可分离变量 · 隐式通解
求 \(\sec^2 x\tan y\,dx + \sec^2 y\tan x\,dy = 0\) 的通解。
✓ 标准解答

Step 1　移项分离（设 \(\tan x\neq 0, \tan y\neq 0\)）：

\[\frac{\sec^2 x}{\tan x}\,dx = -\frac{\sec^2 y}{\tan y}\,dy\]

Step 2　两边积分。注意到 \((\ln|\tan x|)' = \sec^2 x/\tan x\)，同理对 \(y\)：

\[\ln|\tan x| = -\ln|\tan y| + C_1\]

Step 3　整理：

\[\ln|\tan x| + \ln|\tan y| = C_1 \quad\Rightarrow\quad \ln|\tan x\tan y| = C_1\]

取指数，令 \(C = \pm e^{C_1}\)（也可为零，见特解检验）：

\[\tan x\tan y = C\]

检验 \(\tan x=0\) 或 \(\tan y=0\)：若 \(\tan x=0\) 即 \(x=k\pi\)，原方程第一项为 0，第二项为 \(\sec^2 y\cdot 0 = 0\)，满足。这对应 \(C=0\) 的情形，已包含。

答案：\(\boxed{\tan x\tan y = C}\)

验证：隐函数求导，\(\sec^2 x\tan y + \tan x\sec^2 y\cdot y' = 0\)，与移项后的原方程一致 ✓

技巧提炼

分离变量时如果两边同除了 \(y\)（或某个含 \(y\) 的因子），一定要回头检验 \(y=0\) 是否也是解。如果 \(y=0\) 不能被通解中的任意常数 \(C\) 取到，就要单独写出，否则可能丢特解。

题型二：齐次方程

标准步骤

1

确认右端可写成 \(f(y/x)\)

常见形式：\(\displaystyle\frac{dy}{dx} = \frac{y^2+xy}{x^2} = \left(\frac{y}{x}\right)^2 + \frac{y}{x}\)。分子分母同次是典型信号。

2

令 \(u = y/x\)，即 \(y = ux\)

则 \(\displaystyle\frac{dy}{dx} = u + x\frac{du}{dx}\)。代入后右边只含 \(u\)。

3

分离变量，解关于 \(u\) 的方程

化为 \(g(u)\,du = f(x)\,dx\) 的形式，两边积分。

4

回代 \(u = y/x\)

得到 \(x,y\) 的关系式，整理为通解。有初始条件时在此代入求 \(C\)。

例题 3 齐次方程 · 分子分母同次
求 \(\displaystyle\frac{dy}{dx} = \frac{x^2+y^2}{xy}\) 的通解。
✓ 标准解答

Step 1　右端分子分母都是二次，可化简为：

\[\frac{dy}{dx} = \frac{x}{y} + \frac{y}{x}\]

Step 2　令 \(u = y/x\)，则 \(y = ux\)，\(y' = u + xu'\)。代入：

\[u + xu' = \frac{1}{u} + u\]

化简得

\[xu' = \frac{1}{u}\]

Step 3　分离变量：

\[u\,du = \frac{dx}{x}\]

两边积分：

\[\frac{u^2}{2} = \ln|x| + C_1\]

即 \(u^2 = 2\ln|x| + C\)（令 \(C=2C_1\)）。

Step 4　回代 \(u = y/x\)：

\[\left(\frac{y}{x}\right)^2 = 2\ln|x| + C\]

整理得

\[y^2 = x^2(2\ln|x| + C)\]

答案：\(\boxed{y^2 = x^2(2\ln|x| + C)}\)

验证：隐函数求导 \(2yy' = 2x(2\ln|x|+C) + x^2\cdot\frac{2}{x} = 2x\cdot\frac{y^2}{x^2} + 2x = \frac{2y^2}{x} + 2x\)。故 \(y' = \frac{y}{x} + \frac{x}{y} = \frac{x^2+y^2}{xy}\) ✓

例题 4 齐次方程 · 带初始条件求特解
求 \(\displaystyle\frac{dy}{dx} = \frac{y}{x} + \left(\frac{y}{x}\right)^3\) 满足 \(y(1)=1\) 的特解。
✓ 标准解答

Step 1–2　令 \(u = y/x\)，则 \(y' = u + xu'\)。代入：

\[u + xu' = u + u^3\]

化简得

\[xu' = u^3\]

Step 3　分离变量（设 \(u\neq 0\)）：

\[u^{-3}\,du = \frac{dx}{x}\]

两边积分：

\[-\frac{1}{2u^2} = \ln|x| + C\]

Step 4　回代 \(u = y/x\)：

\[-\frac{x^2}{2y^2} = \ln|x| + C\]

由初始条件 \(y(1)=1\)，代入 \(x=1, y=1\)：

\[-\frac{1}{2} = 0 + C \quad\Rightarrow\quad C = -\frac{1}{2}\]

于是

\[-\frac{x^2}{2y^2} = \ln|x| - \frac{1}{2}\]

整理（注意 \(x>0\) 在初始点附近）：

\[\frac{x^2}{y^2} = 1 - 2\ln x \quad\Rightarrow\quad y^2 = \frac{x^2}{1-2\ln x}\]

因 \(y(1)=1>0\)，取正根：

\[y = \frac{x}{\sqrt{1-2\ln x}}\]

检验 \(u=0\) 即 \(y=0\)：代入原方程，左边 \(y'=0\)，右边 \(0+0=0\)，但 \(y(1)=1\neq 0\)，不影响本特解。

答案：\(\boxed{y = \dfrac{x}{\sqrt{1-2\ln x}}}\)

验证：令 \(y = x(1-2\ln x)^{-1/2}\)，则

\[y' = (1-2\ln x)^{-1/2} + x\cdot\left(-\frac{1}{2}\right)(1-2\ln x)^{-3/2}\cdot\left(-\frac{2}{x}\right) = (1-2\ln x)^{-1/2} + (1-2\ln x)^{-3/2}\]

而 \(\dfrac{y}{x}+\left(\dfrac{y}{x}\right)^3 = (1-2\ln x)^{-1/2} + (1-2\ln x)^{-3/2} = y'\) ✓

齐次方程的识别信号

如果方程右端分子分母的每一项关于 \(x,y\) 的总次数相同（如同为一次、同为二次），十有八九是齐次方程。先尝试把右端整理成 \(f(y/x)\) 的形式，如果成功，就令 \(u=y/x\)。

题型三：一阶线性方程

标准步骤

1

标准化为 \(y'+P(x)y=Q(x)\)

确保 \(y'\) 的系数为 1。如果不是，方程两边同除该系数。

2

求积分因子 \(\mu(x) = e^{\int P(x)\,dx}\)

只需求一个原函数，积分常数取 0。

3

套通解公式

\[y = e^{-\int P\,dx}\left[\int Q(x)\,e^{\int P\,dx}\,dx + C\right]\]
4

有初始条件时求 \(C\)，得到特解

例题 5 一阶线性 · 基本型
求 \(\displaystyle\frac{dy}{dx} - \frac{2}{x}y = x^2\) 的通解。
✓ 标准解答

Step 1　已是标准形式，\(P(x)=-\dfrac{2}{x}\)，\(Q(x)=x^2\)。

Step 2　求积分因子：

\[\int P(x)\,dx = \int\left(-\frac{2}{x}\right)dx = -2\ln|x| = \ln\frac{1}{x^2}\]
\[\mu(x) = e^{\int P\,dx} = \frac{1}{x^2}\]

Step 3　套公式。先算 \(e^{-\int P\,dx} = x^2\)，再算

\[\int Q(x)\mu(x)\,dx = \int x^2\cdot\frac{1}{x^2}\,dx = \int dx = x\]

于是

\[y = x^2(x + C) = x^3 + Cx^2\]

答案：\(\boxed{y = x^3 + Cx^2}\)

验证：\(y' = 3x^2 + 2Cx\)。代入左边：\(y'-\dfrac{2}{x}y = 3x^2+2Cx - \dfrac{2}{x}(x^3+Cx^2) = 3x^2+2Cx-2x^2-2Cx = x^2\) ✓

例题 6 一阶线性 · 标准化后求解（特解）
求 \(xy'+y = \sin x\) 满足 \(y(\pi)=\dfrac{1}{\pi}\) 的特解。
✓ 标准解答

Step 1　标准化：两边同除以 \(x\)（\(x\neq 0\)），得

\[y' + \frac{1}{x}y = \frac{\sin x}{x}\]

其中 \(P(x)=\dfrac{1}{x}\)，\(Q(x)=\dfrac{\sin x}{x}\)。

Step 2　积分因子：

\[\int P(x)\,dx = \ln|x|, \quad \mu(x) = e^{\ln|x|} = |x|\]

在 \(x>0\) 的邻域内可取 \(\mu(x)=x\)（在 \(x<0\) 时取 \(-x\)，结果等价，只影响绝对值处理）。

Step 3　套公式。\(e^{-\int P\,dx} = \dfrac{1}{x}\)，

\[\int Q(x)\mu(x)\,dx = \int\frac{\sin x}{x}\cdot x\,dx = \int\sin x\,dx = -\cos x + C\]

故通解为

\[y = \frac{1}{x}(-\cos x + C) = \frac{C - \cos x}{x}\]

Step 4　代入 \(y(\pi)=\dfrac{1}{\pi}\)：

\[\frac{1}{\pi} = \frac{C - \cos\pi}{\pi} = \frac{C + 1}{\pi}\]

解得 \(C+1=1\)，即 \(C=0\)。

答案：\(\boxed{y = -\dfrac{\cos x}{x}}\)

验证：\(y' = \dfrac{x\sin x + \cos x}{x^2} = \dfrac{\sin x}{x} + \dfrac{\cos x}{x^2}\)。代入 \(xy'+y = x\left(\dfrac{\sin x}{x}+\dfrac{\cos x}{x^2}\right) + \left(-\dfrac{\cos x}{x}\right) = \sin x + \dfrac{\cos x}{x} - \dfrac{\cos x}{x} = \sin x\) ✓

且 \(y(\pi) = -\dfrac{-1}{\pi} = \dfrac{1}{\pi}\) ✓

一阶线性的快捷观察

有些一阶线性方程左边恰好是某个乘积的导数，可一步到位。例如例题 6 中 \(xy'+y = (xy)'\)，所以原方程等价于 \((xy)' = \sin x\)，直接积分得 \(xy = -\cos x + C\)。考场上如果一眼看出这种结构，能省不少时间。

题型四：伯努利方程

标准步骤

1

确认方程为 \(y'+P(x)y=Q(x)y^n\)，其中 \(n\neq 0,1\)

当 \(n=0\) 或 \(n=1\) 时，方程退化为普通一阶线性，不需要伯努利代换。

2

令 \(z = y^{1-n}\)

则 \(z' = (1-n)y^{-n}y'\)。方程两边同乘 \((1-n)y^{-n}\) 可化为关于 \(z\) 的一阶线性方程。

3

解关于 \(z\) 的一阶线性方程

用积分因子法求出 \(z = \varphi(x,C)\)。

4

回代 \(z = y^{1-n}\)，解出 \(y\)

得到 \(y = \bigl[\varphi(x,C)\bigr]^{\frac{1}{1-n}}\)。

例题 7 伯努利方程 · 基本型（\(n=2\)）
求 \(\displaystyle\frac{dy}{dx} + y = xy^2\) 的通解。
✓ 标准解答

Step 1　方程形如 \(y'+P(x)y=Q(x)y^n\)，其中 \(n=2\)（\(n\neq 0,1\)），是伯努利方程。

Step 2　令 \(z = y^{1-2} = y^{-1} = \dfrac{1}{y}\)。则 \(z' = -y^{-2}y'\)。原方程两边同乘 \((-y^{-2})\)：

\[-y^{-2}y' - y^{-1} = -x\]

即

\[z' - z = -x\]

Step 3　关于 \(z\) 的一阶线性方程：\(z'+(-1)z=-x\)。积分因子

\[\mu(x) = e^{\int(-1)\,dx} = e^{-x}\]

套公式：

\[z = e^{x}\left[\int(-x)e^{-x}\,dx + C\right]\]

对 \(\int(-x)e^{-x}\,dx\) 用分部积分（\(u=-x, dv=e^{-x}dx\)）：

\[\int(-x)e^{-x}\,dx = xe^{-x} - \int e^{-x}\,dx = xe^{-x} + e^{-x} = e^{-x}(x+1)\]

故

\[z = e^{x}\bigl[e^{-x}(x+1) + C\bigr] = x + 1 + Ce^{x}\]

Step 4　回代 \(z = \dfrac{1}{y}\)：

\[\frac{1}{y} = x + 1 + Ce^{x}\]

解得

\[y = \frac{1}{x + 1 + Ce^{x}}\]

检验 \(y=0\)：代入原方程左边 \(y'=0\)，右边 \(0+0=0\)，满足，但 \(y=0\) 不能由上述通解取某个 \(C\) 得到（因为分母有限时 \(y\neq 0\)），所以 \(y=0\) 是一个漏掉的特解。不过在求通解时，通常默认 \(y\neq 0\) 进行推导，最后写出的含任意常数 \(C\) 的解即为通解。完整答案应包含 \(y=0\) 或注明 \(C\) 为任意常数时 \(y\) 的表达式。

答案：\(\boxed{y = \dfrac{1}{x + 1 + Ce^{x}}}\)（以及特解 \(y=0\)）

验证：由 \(z = x+1+Ce^x\)，有 \(z' = 1+Ce^x\)。则 \(z'-z = 1+Ce^x-(x+1+Ce^x) = -x\)。又 \(z'-z = -y'/y^2 - 1/y\)，故 \(-y'/y^2-1/y=-x\)，即 \(y'/y^2+1/y=x\)，两边乘 \(y^2\) 得 \(y'+y=xy^2\) ✓

例题 8 伯努利方程 · \(n=2\) 化线性
求 \(\displaystyle\frac{dy}{dx} + \frac{y}{x} = y^2\) 的通解。
✓ 标准解答

Step 1　\(n=2\)，伯努利方程。

Step 2　令 \(z = y^{-1}\)，则 \(z' = -y^{-2}y'\)。原方程两边同乘 \((-y^{-2})\)：

\[-y^{-2}y' - \frac{1}{xy} = -1\]

即

\[z' - \frac{1}{x}z = -1\]

Step 3　关于 \(z\) 的线性方程：\(P(x)=-\dfrac{1}{x}\)。积分因子

\[\mu(x) = e^{\int(-1/x)\,dx} = e^{-\ln|x|} = \frac{1}{|x|}\]

在 \(x>0\) 时取 \(\mu(x)=\dfrac{1}{x}\)。\(e^{-\int P\,dx} = x\)，

\[\int Q(x)\mu(x)\,dx = \int(-1)\cdot\frac{1}{x}\,dx = -\ln|x| + C\]

于是

\[z = x(-\ln|x| + C) = Cx - x\ln|x|\]

Step 4　回代 \(z = \dfrac{1}{y}\)：

\[\frac{1}{y} = Cx - x\ln|x|\]

解得

\[y = \frac{1}{Cx - x\ln|x|}\]

检验 \(y=0\)：满足原方程，但不能从通解取常数得到，为独立特解。

答案：\(\boxed{y = \dfrac{1}{Cx - x\ln|x|}}\)（以及特解 \(y=0\)）

验证：\(z = Cx - x\ln|x|\)，\(z' = C - \ln|x| - 1\)。则 \(z'-\dfrac{z}{x} = C-\ln|x|-1 - (C-\ln|x|) = -1\)。又 \(z'-\dfrac{z}{x} = -\dfrac{y'}{y^2} - \dfrac{1}{xy}\)，故 \(-y'/y^2-1/(xy)=-1\)，即 \(y'/y^2+1/(xy)=1\)，两边乘 \(y^2\) 得 \(y'+y/x=y^2\) ✓

伯努利方程的代换要点

令 \(z = y^{1-n}\) 后，新方程一定是关于 \(z\) 的一阶线性方程。如果代换后不是线性的，说明你同乘的系数错了——正确的做法是把原方程两边除以 \(y^n\)，再把 \(y'/y^n\) 凑成 \(z'/(1-n)\)。

⚠️ 常见错误

错误 1：一阶线性公式符号搞反

通解公式是 \(y = e^{-\int P\,dx}\left[\int Q e^{\int P\,dx}dx + C\right]\)。很多同学把外层的 \(e^{-\int P\,dx}\) 和内层的 \(e^{\int P\,dx}\) 写反，或者符号记混。记住：外层是负号（把积分因子"消掉"），内层是正号（把 \(Q\) 乘上积分因子）。

错误 2：齐次方程代换后忘记回代 \(u=y/x\)

令 \(u=y/x\) 解出 \(u\) 的表达式后，最后必须回代成 \(x,y\) 的关系。如果答案里还留着 \(u\)，判卷时直接判错。回代时把 \(u\) 全部替换为 \(y/x\)，再整理。

错误 3：分离变量时漏掉积分常数，或写了两个常数

两边积分时，理论上各有一个常数：\(\int g(y)dy = G(y)+C_1\)，\(\int f(x)dx = F(x)+C_2\)。合并时只写一个 \(C = C_2-C_1\) 即可。写两个常数是画蛇添足，漏写常数则是根本性错误。

错误 4：伯努利代换时 \(n\) 搞错，或忘了 \(n\neq 0,1\)

伯努利方程的标准形式是 \(y'+P(x)y=Q(x)y^n\)。如果 \(n=0\)，方程就是 \(y'+P(x)y=Q(x)\)，普通线性；如果 \(n=1\)，方程就是 \(y'+[P(x)-Q(x)]y=0\)，可分离变量。只有当 \(n\neq 0,1\) 时才需要令 \(z=y^{1-n}\)。

错误 5：分离变量时同除因子，漏掉使该因子为零的特解

例如解 \(y'=xy\) 时两边除以 \(y\)，就默认了 \(y\neq 0\)。但 \(y=0\) 代入原方程也成立。如果通解 \(y=Ce^{x^2/2}\) 中的 \(C\) 可以取到 0，则 \(y=0\) 已被包含；如果不能，则需单独写出。伯努利方程中 \(y=0\) 常常是这种漏掉的特解。
