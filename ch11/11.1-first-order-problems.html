11.1 · 题型

一阶方程 标准步骤与例题

3秒识别类型，然后套公式。

题型：识别 + 套解法

识别顺序

可分离 → 齐次 → 线性 → 伯努利

例题 1：可分离变量

例题可分离
求 \( rac{dy}{dx} = xy\) 的通解。
✓ 标准解答

分离：\( rac{dy}{y} = x\,dx\)

积分：\(\ln|y| = rac{x^2}{2} + C_1\)

\(y = Ce^{x^2/2}\)

例题 2：一阶线性

例题一阶线性
求 \( rac{dy}{dx} + 2xy = x\) 的通解。
✓ 标准解答

\(P(x)=2x, Q(x)=x\)，\(I = \int 2x\,dx = x^2\)

\(y = e^{-x^2}\left( rac{1}{2}e^{x^2} + C ight) = rac{1}{2} + Ce^{-x^2}\)
常见错误

一阶线性方程套公式时，\(e^{\int Pdx}\) 和 \(e^{-\int Pdx}\) 不要搞反。
