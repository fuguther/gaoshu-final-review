二阶线性微分方程 习题全解

《高等数学（生化医农类）下册》习题 11.2 二阶常系数齐次 / 非齐次方程求解 · 考场标准步骤

高数C 期末 Q3（30 分）的核心——二阶常系数线性方程。本页收齐习题 11.2：齐次/高阶求通解（问题1）、齐次特解（问题2）、非齐次待定系数求通解含共振（问题3）、非齐次特解（问题4）共 40 小题，A 路双解（此类为标准特征根法，答案规范可循）。

1.求通解

(1)\(\displaystyle \frac{d^{2}x}{dt^{2}}-\frac{dx}{dt}-2x=0\)
特征方程为 \[ r^2 - r - 2 = 0 \]
因式分解得 \[ (r-2)(r+1) = 0 \]
解得特征根 \[ r_1 = 2, \quad r_2 = -1 \]
通解为 \[ x = C_1 e^{2t} + C_2 e^{-t} \]
答案\( x = C_1 e^{2t} + C_2 e^{-t} \)
(2)\(\displaystyle \frac{d^{2}y}{dx^{2}}-4\frac{dy}{dx}+3y=0\)
特征方程为 \[r^{2}-4r+3=0\]
因式分解得 \[(r-1)(r-3)=0\]
解得特征根 \[r_{1}=1,\quad r_{2}=3\]
由于特征根为两个不相等的实根，通解为 \[y=C_{1}e^{r_{1}x}+C_{2}e^{r_{2}x}\]
代入特征根得 \[y=C_{1}e^{x}+C_{2}e^{3x}\]
答案\[y=C_{1}e^{x}+C_{2}e^{3x}\]
(3)\(\displaystyle \frac{d^{2}s}{dt^{2}}-2\frac{ds}{dt}+s=0\)
特征方程为 \[ r^2 - 2r + 1 = 0 \]
因式分解得 \[ (r-1)^2 = 0 \]
解得二重特征根 \[ r_1 = r_2 = 1 \]
对于二重实根，通解形式为 \[ s = (C_1 + C_2 t)e^{rt} \]
代入 \( r = 1 \)，得通解 \[ s = (C_1 + C_2 t)e^{t} \]
答案\( s = (C_1 + C_2 t)e^{t} \)
(4)\(\displaystyle \frac{d^{2}s}{dt^{2}}+2\frac{ds}{dt}+2s=0\)
特征方程为 \(r^{2}+2r+2=0\)
解得 \(r=\frac{-2\pm\sqrt{4-8}}{2}=\frac{-2\pm\sqrt{-4}}{2}=\frac{-2\pm 2i}{2}=-1\pm i\)
即 \(r_{1}=-1+i\), \(r_{2}=-1-i\)
通解为 \(s=e^{-t}(C_{1}\cos t+C_{2}\sin t)\)
答案\(s=e^{-t}(C_{1}\cos t+C_{2}\sin t)\)
(5)\(\displaystyle \frac{d^{2}s}{dt^{2}}-2\frac{ds}{dt}+5s=0\)
特征方程为 \[r^{2}-2r+5=0\]
解得 \[r=\frac{2\pm\sqrt{4-20}}{2}=\frac{2\pm\sqrt{-16}}{2}=\frac{2\pm 4i}{2}=1\pm 2i\]
即 \(\alpha=1\), \(\beta=2\)
通解为 \[s=e^{t}(C_{1}\cos 2t+C_{2}\sin 2t)\]
答案\[s=e^{t}(C_{1}\cos 2t+C_{2}\sin 2t)\]
(6)\(\displaystyle y''-4y'=0\)
特征方程为 \[r^2 - 4r = 0\]
因式分解得 \[r(r-4) = 0\]
解得特征根 \[r_1 = 0, \quad r_2 = 4\]
通解为 \[y = C_1 e^{0 \cdot x} + C_2 e^{4x}\]
即 \[y = C_1 + C_2 e^{4x}\]
答案\[y = C_1 + C_2 e^{4x}\]
(7)\(\displaystyle y''-2y'+(1-a^{2})y=0\ (a>0)\)
\[特征方程为\]
\[求根\]
由于 \( a>0 \)，故 \( r_{1}=1+a \)，\( r_{2}=1-a \) 为两个不等实根
\[通解为\]
答案\( y=C_{1}e^{(1+a)x}+C_{2}e^{(1-a)x} \)
(8)\(\displaystyle y''-10y'+34y=0\)
特征方程为 \(r^2 - 10r + 34 = 0\)
求根：\(r = \frac{10 \pm \sqrt{100 - 136}}{2} = \frac{10 \pm \sqrt{-36}}{2} = \frac{10 \pm 6i}{2} = 5 \pm 3i\)
通解为 \(y = e^{5x}(C_1 \cos 3x + C_2 \sin 3x)\)
答案\(y = e^{5x}(C_1 \cos 3x + C_2 \sin 3x)\)
(9)\(\displaystyle y''-4y'+5y=0\)
特征方程为 \( r^2 - 4r + 5 = 0 \)
解得 \( r = \frac{4 \pm \sqrt{16-20}}{2} = \frac{4 \pm \sqrt{-4}}{2} = \frac{4 \pm 2i}{2} = 2 \pm i \)
即 \( \alpha = 2 \), \( \beta = 1 \)
通解为 \( y = e^{2x}(C_1 \cos x + C_2 \sin x) \)
答案\( y = e^{2x}(C_1 \cos x + C_2 \sin x) \)
(10)\(\displaystyle y''-6y'+11y=0\)
特征方程为 \[r^2 - 6r + 11 = 0\]
求根公式得 \[r = \frac{6 \pm \sqrt{36 - 44}}{2} = \frac{6 \pm \sqrt{-8}}{2} = \frac{6 \pm 2\sqrt{2}i}{2} = 3 \pm \sqrt{2}i\]
即 \(\alpha = 3\), \(\beta = \sqrt{2}\)
通解为 \[y = e^{3x}\left(C_1 \cos \sqrt{2}x + C_2 \sin \sqrt{2}x\right)\]
答案\[y = e^{3x}\left(C_1 \cos \sqrt{2}x + C_2 \sin \sqrt{2}x\right)\]
(11)\(\displaystyle y''-y=0\)
特征方程为 \(r^2 - 1 = 0\)
解得 \(r^2 = 1\)，即 \(r_1 = 1\)，\(r_2 = -1\)
两个不相等的实根，通解为 \(y = C_1 e^{r_1 x} + C_2 e^{r_2 x}\)
代入得 \(y = C_1 e^{x} + C_2 e^{-x}\)
答案\(y = C_1 e^{x} + C_2 e^{-x}\)
(12)\(\displaystyle y^{(4)}+8y''+16y=0\)
特征方程为 \( r^4 + 8r^2 + 16 = 0 \)
令 \( u = r^2 \)，得 \( u^2 + 8u + 16 = 0 \)，即 \( (u+4)^2 = 0 \)
解得 \( u = -4 \)（二重根），故 \( r^2 = -4 \)
由 \( r^2 = -4 \)，得 \( r = \pm 2i \)，其中 \( r = 2i \) 为二重根，\( r = -2i \) 为二重根
对于二重共轭复根 \( \alpha \pm \beta i \)（此处 \( \alpha = 0, \beta = 2 \)），通解形式为
代入 \( \alpha = 0, \beta = 2 \)，得
答案\[ y = (C_1 + C_2 x)\cos 2x + (C_3 + C_4 x)\sin 2x \]
(13)\(\displaystyle y'''-3y''+9y'+13y=0\)
特征方程为 \[r^3 - 3r^2 + 9r + 13 = 0\]
试根，当 \(r = -1\) 时：\[(-1)^3 - 3(-1)^2 + 9(-1) + 13 = -1 - 3 - 9 + 13 = 0\]，故 \(r = -1\) 是根
多项式除法：\[(r^3 - 3r^2 + 9r + 13) \div (r + 1) = r^2 - 4r + 13\]
验证：\((r+1)(r^2-4r+13) = r^3 - 4r^2 + 13r + r^2 - 4r + 13 = r^3 - 3r^2 + 9r + 13\)
对 \(r^2 - 4r + 13 = 0\) 用求根公式：\[r = \frac{4 \pm \sqrt{16-52}}{2} = \frac{4 \pm \sqrt{-36}}{2} = \frac{4 \pm 6i}{2} = 2 \pm 3i\]
三个特征根为 \(r_1 = -1\)，\(r_2 = 2 + 3i\)，\(r_3 = 2 - 3i\)
对应基本解：\(e^{-x}\)，\(e^{2x}\cos 3x\)，\(e^{2x}\sin 3x\)
通解为 \[y = C_1 e^{-x} + e^{2x}(C_2 \cos 3x + C_3 \sin 3x)\]
答案\(y = C_1 e^{-x} + e^{2x}(C_2 \cos 3x + C_3 \sin 3x)\)，其中 \(C_1, C_2, C_3\) 为任意常数
(14)\(\displaystyle y^{(4)}-8y'=0\)
特征方程为 \( r^4 - 8r = 0 \)
因式分解得 \( r(r^3 - 8) = 0 \)
即 \( r(r-2)(r^2+2r+4) = 0 \)
由 \( r = 0 \) 得 \( r_1 = 0 \)
由 \( r - 2 = 0 \) 得 \( r_2 = 2 \)
由 \( r^2 + 2r + 4 = 0 \) 得 \( r = \frac{-2 \pm \sqrt{4-16}}{2} = \frac{-2 \pm \sqrt{-12}}{2} = \frac{-2 \pm 2\sqrt{3}i}{2} = -1 \pm \sqrt{3}i \)
即 \( r_3 = -1 + \sqrt{3}i \)，\( r_4 = -1 - \sqrt{3}i \)
对应实根 \( r_1 = 0 \) 的解为 \( y_1 = e^{0 \cdot x} = 1 \)
对应实根 \( r_2 = 2 \) 的解为 \( y_2 = e^{2x} \)
对应共轭复根 \( r_{3,4} = -1 \pm \sqrt{3}i \) 的解为 \( y_3 = e^{-x}\cos(\sqrt{3}x) \)，\( y_4 = e^{-x}\sin(\sqrt{3}x) \)
通解为 \( y = C_1 \cdot 1 + C_2 e^{2x} + C_3 e^{-x}\cos(\sqrt{3}x) + C_4 e^{-x}\sin(\sqrt{3}x) \)
答案\( y = C_1 + C_2 e^{2x} + e^{-x}\left(C_3\cos\sqrt{3}x + C_4\sin\sqrt{3}x\right) \)

2.求特解

(1)\(\displaystyle \frac{d^{2}s}{dt^{2}}+3\frac{ds}{dt}+2s=0, 当 t=0 时, s=0, \frac{ds}{dt}=1\)
特征方程为 \(r^2 + 3r + 2 = 0\)
因式分解得 \((r+1)(r+2) = 0\)
解得特征根 \(r_1 = -1\), \(r_2 = -2\)
通解为 \(s = C_1 e^{-t} + C_2 e^{-2t}\)
求导得 \(\frac{ds}{dt} = -C_1 e^{-t} - 2C_2 e^{-2t}\)
代入初始条件 \(s(0) = 0\): \(C_1 + C_2 = 0\)
代入初始条件 \(\frac{ds}{dt}\big|_{t=0} = 1\): \(-C_1 - 2C_2 = 1\)
由 \(C_1 + C_2 = 0\) 得 \(C_1 = -C_2\)
代入 \(-C_1 - 2C_2 = 1\): \(-(-C_2) - 2C_2 = 1\), 即 \(C_2 - 2C_2 = 1\)
解得 \(-C_2 = 1\), 故 \(C_2 = -1\), \(C_1 = 1\)
特解为 \(s = e^{-t} - e^{-2t}\)
答案\(s = e^{-t} - e^{-2t}\)
(2)\(\displaystyle \frac{d^{2}x}{dt^{2}}+n^{2}x=0, 当 t=0 时, x=a, \frac{dx}{dt}=0\)
特征方程为 \(r^2 + n^2 = 0\)
解得 \(r = \pm ni\)
通解为 \(x = C_1 \cos nt + C_2 \sin nt\)
求导得 \(\frac{dx}{dt} = -nC_1 \sin nt + nC_2 \cos nt\)
代入 \(t = 0\), \(x = a\): \(C_1 \cos 0 + C_2 \sin 0 = a\), 即 \(C_1 = a\)
代入 \(t = 0\), \(\frac{dx}{dt} = 0\): \(-nC_1 \sin 0 + nC_2 \cos 0 = 0\), 即 \(nC_2 = 0\), 故 \(C_2 = 0\)
特解为 \(x = a \cos nt\)
答案\(x = a \cos nt\)
(3)\(\displaystyle \frac{d^{2}x}{dt^{2}}-n^{2}x=0, 当 t=0 时, x=2, \frac{dx}{dt}=0\)
特征方程为 \[r^{2}-n^{2}=0\]
解得 \[r_{1}=n,\quad r_{2}=-n\]
通解为 \[x(t)=C_{1}e^{nt}+C_{2}e^{-nt}\]
求导得 \[\frac{dx}{dt}=nC_{1}e^{nt}-nC_{2}e^{-nt}\]
代入初始条件 \(t=0\), \(x=2\): \[C_{1}+C_{2}=2\]
代入初始条件 \(t=0\), \(\frac{dx}{dt}=0\): \[nC_{1}-nC_{2}=0\] 即 \[C_{1}-C_{2}=0\]
联立解得 \[C_{1}=1,\quad C_{2}=1\]
特解为 \[x(t)=e^{nt}+e^{-nt}=2\cosh(nt)\]
答案\[x(t)=e^{nt}+e^{-nt}\] 或等价地 \[x(t)=2\cosh(nt)\]
(4)\(\displaystyle \frac{d^{2}x}{dt^{2}}-a\frac{dx}{dt}=0, 当 t=0 时, x=0, \frac{dx}{dt}=a\)
特征方程为 \[r^{2}-ar=0\]
解得 \[r(r-a)=0,\quad r_{1}=0,\quad r_{2}=a\]
通解为 \[x=C_{1}+C_{2}e^{at}\]
求导得 \[\frac{dx}{dt}=C_{2}ae^{at}\]
代入初始条件 \(t=0,\ x=0\)： \[C_{1}+C_{2}=0\]
代入初始条件 \(t=0,\ \frac{dx}{dt}=a\)： \[C_{2}a=a,\quad C_{2}=1\]
由 \(C_{1}+C_{2}=0\) 得 \[C_{1}=-1\]
特解为 \[x=-1+e^{at}\]
答案\[x=e^{at}-1\]
(5)\(\displaystyle \frac{d^{2}s}{dt^{2}}+8\frac{ds}{dt}+25s=0, 当 t=0 时, s=4, \frac{ds}{dt}=-16\)
特征方程为 \[r^{2}+8r+25=0\]
解得 \[r=\frac{-8\pm\sqrt{64-100}}{2}=\frac{-8\pm\sqrt{-36}}{2}=\frac{-8\pm 6i}{2}=-4\pm 3i\]
通解为 \[s=e^{-4t}\left(C_{1}\cos 3t+C_{2}\sin 3t\right)\]
求导得 \[\frac{ds}{dt}=-4e^{-4t}\left(C_{1}\cos 3t+C_{2}\sin 3t\right)+e^{-4t}\left(-3C_{1}\sin 3t+3C_{2}\cos 3t\right)\] \[=e^{-4t}\left[\left(-4C_{1}+3C_{2}\right)\cos 3t+\left(-4C_{2}-3C_{1}\right)\sin 3t\right]\]
代入 \(t=0\), \(s=4\): \[e^{0}\left(C_{1}\cos 0+C_{2}\sin 0\right)=C_{1}=4\]
代入 \(t=0\), \(\frac{ds}{dt}=-16\): \[e^{0}\left[\left(-4C_{1}+3C_{2}\right)\cos 0+\left(-4C_{2}-3C_{1}\right)\sin 0\right]=-4C_{1}+3C_{2}=-16\]
将 \(C_{1}=4\) 代入 \(-4\times 4+3C_{2}=-16\), 得 \[-16+3C_{2}=-16\], \[3C_{2}=0\], \[C_{2}=0\]
特解为 \[s=4e^{-4t}\cos 3t\]
答案\[s=4e^{-4t}\cos 3t\]
(6)\(\displaystyle \frac{d^{2}x}{dt^{2}}-6\frac{dx}{dt}+10x=0, 当 t=0 时, x=1, \frac{dx}{dt}=4\)
特征方程为 \(r^2 - 6r + 10 = 0\)
解得 \(r = \frac{6 \pm \sqrt{36-40}}{2} = \frac{6 \pm \sqrt{-4}}{2} = \frac{6 \pm 2i}{2} = 3 \pm i\)
通解为 \(x = e^{3t}(C_1 \cos t + C_2 \sin t)\)
求导得 \(\frac{dx}{dt} = 3e^{3t}(C_1 \cos t + C_2 \sin t) + e^{3t}(-C_1 \sin t + C_2 \cos t) = e^{3t}[(3C_1+C_2)\cos t + (3C_2-C_1)\sin t]\)
代入初值 \(t=0, x=1\): \(e^{0}(C_1 \cos 0 + C_2 \sin 0) = C_1 = 1\)
代入初值 \(t=0, \frac{dx}{dt}=4\): \(e^{0}[(3C_1+C_2)\cos 0 + (3C_2-C_1)\sin 0] = 3C_1 + C_2 = 4\)
由 \(C_1 = 1\) 得 \(3 \times 1 + C_2 = 4\), 故 \(C_2 = 1\)
特解为 \(x = e^{3t}(\cos t + \sin t)\)
答案\(x = e^{3t}(\cos t + \sin t)\)
(7)\(\displaystyle 2y''+3y=2\sqrt{6}\ y', y\big|_{x=0}=0, y'\big|_{x=0}=1\)
方程化为标准形式：\(2y'' - 2\sqrt{6}y' + 3y = 0\)
特征方程：\(2r^2 - 2\sqrt{6}r + 3 = 0\)
求根：\[r = \frac{2\sqrt{6} \pm \sqrt{24 - 24}}{4} = \frac{2\sqrt{6}}{4} = \frac{\sqrt{6}}{2}\]
重根 \(r = \frac{\sqrt{6}}{2}\)，通解为：\[y = (C_1 + C_2 x)e^{\frac{\sqrt{6}}{2}x}\]
求导：\[y' = C_2 e^{\frac{\sqrt{6}}{2}x} + (C_1 + C_2 x)\frac{\sqrt{6}}{2}e^{\frac{\sqrt{6}}{2}x} = \left(C_2 + \frac{\sqrt{6}}{2}C_1 + \frac{\sqrt{6}}{2}C_2 x\right)e^{\frac{\sqrt{6}}{2}x}\]
代入 \(y(0) = 0\)：\(C_1 = 0\)
代入 \(y'(0) = 1\)：\(C_2 + \frac{\sqrt{6}}{2} \cdot 0 = 1\)，即 \(C_2 = 1\)
特解为：\[y = xe^{\frac{\sqrt{6}}{2}x}\]
答案\(y = xe^{\frac{\sqrt{6}}{2}x}\)

3.求通解

(1)\(\displaystyle y''+y=ae^{2x}\)
特征方程为 \(r^2+1=0\)，解得 \(r=\pm i\)
齐次通解为 \(y_h=C_1\cos x+C_2\sin x\)
设特解形式为 \(y_p=be^{2x}\)
代入方程：\(y_p'=2be^{2x}\)，\(y_p''=4be^{2x}\)
\(4be^{2x}+be^{2x}=ae^{2x}\)
\(5b=a\)，得 \(b=\frac{a}{5}\)
特解为 \(y_p=\frac{a}{5}e^{2x}\)
通解为 \(y=y_h+y_p=C_1\cos x+C_2\sin x+\frac{a}{5}e^{2x}\)
答案\(y=C_1\cos x+C_2\sin x+\frac{a}{5}e^{2x}\)
(2)\(\displaystyle y''+y=4\cos x\)
求齐次方程 \(y''+y=0\) 的通解。特征方程为 \(r^2+1=0\)，解得 \(r=\pm i\)。齐次通解为 \(y_h=C_1\cos x+C_2\sin x\)。
求非齐次方程的特解。由于右端 \(4\cos x\) 与齐次解中的 \(\cos x\) 共振，设特解形式为 \(y_p=Ax\cos x+Bx\sin x\)。
计算 \(y_p'\)：
计算 \(y_p''\)：
代入方程 \(y_p''+y_p=4\cos x\)：
\[整理得：\]
比较系数：\(2B=4\)，\(-2A=0\)，解得 \(A=0\)，\(B=2\)。
特解为 \(y_p=2x\sin x\)。
通解为 \(y=y_h+y_p=C_1\cos x+C_2\sin x+2x\sin x\)。
答案\(y=C_1\cos x+C_2\sin x+2x\sin x\)
(3)\(\displaystyle y''+y=4\sin 2x\)
求齐次方程 \( y''+y=0 \) 的通解。特征方程为 \( r^2+1=0 \)，解得 \( r=\pm i \)。齐次通解为 \( y_h=C_1\cos x+C_2\sin x \)。
设非齐次方程特解。由于右端为 \( 4\sin 2x \)，而 \( 2i \) 不是特征根，设 \( y_p=A\cos 2x+B\sin 2x \)。
计算导数：\( y_p'=-2A\sin 2x+2B\cos 2x \)，\( y_p''=-4A\cos 2x-4B\sin 2x \)。
代入方程 \( y_p''+y_p=4\sin 2x \)：
比较系数：\( -3A=0 \)，\( -3B=4 \)。解得 \( A=0 \)，\( B=-\frac{4}{3} \)。
特解为 \( y_p=-\frac{4}{3}\sin 2x \)。
答案通解为 \( y=C_1\cos x+C_2\sin x-\frac{4}{3}\sin 2x \)
(4)\(\displaystyle y''+9y=5x^{2}\)
齐次方程 \(y''+9y=0\) 的特征方程为 \[r^2+9=0\]
解得 \[r=\pm 3i\]
齐次通解为 \[Y=C_1\cos 3x+C_2\sin 3x\]
设特解形式为 \[y^*=ax^2+bx+c\]
求导得 \[y^{*'}=2ax+b,\quad y^{*''}=2a\]
代入原方程：\[2a+9(ax^2+bx+c)=5x^2\]
整理得 \[9ax^2+9bx+(2a+9c)=5x^2\]
比较系数：\[9a=5,\quad 9b=0,\quad 2a+9c=0\]
解得 \[a=\frac{5}{9},\quad b=0,\quad c=-\frac{10}{81}\]
特解为 \[y^*=\frac{5}{9}x^2-\frac{10}{81}\]
通解为 \[y=C_1\cos 3x+C_2\sin 3x+\frac{5}{9}x^2-\frac{10}{81}\]
答案\(y=C_1\cos 3x+C_2\sin 3x+\frac{5}{9}x^2-\frac{10}{81}\)
(5)\(\displaystyle y''-7y'+6y=\sin x\)
特征方程为 \(r^2-7r+6=0\)，即 \((r-1)(r-6)=0\)，得 \(r_1=1, r_2=6\)
齐次通解为 \(Y=C_1e^x+C_2e^{6x}\)
设特解 \(y^*=A\cos x+B\sin x\)
则 \(y^{*'}=-A\sin x+B\cos x\)，\(y^{*''}=-A\cos x-B\sin x\)
代入原方程：\((-A\cos x-B\sin x)-7(-A\sin x+B\cos x)+6(A\cos x+B\sin x)=\sin x\)
整理：\((-A-7B+6A)\cos x+(-B+7A+6B)\sin x=\sin x\)
即 \((5A-7B)\cos x+(7A+5B)\sin x=\sin x\)
比较系数：\(\begin{cases}5A-7B=0\\7A+5B=1\end{cases}\)
由第一式 \(A=\frac{7B}{5}\)，代入第二式：\(7\cdot\frac{7B}{5}+5B=1\)，即 \(\frac{49B+25B}{5}=1\)，\(\frac{74B}{5}=1\)
得 \(B=\frac{5}{74}\)，\(A=\frac{7}{5}\cdot\frac{5}{74}=\frac{7}{74}\)
特解为 \(y^*=\frac{7}{74}\cos x+\frac{5}{74}\sin x\)
通解为 \(y=C_1e^x+C_2e^{6x}+\frac{7}{74}\cos x+\frac{5}{74}\sin x\)
答案\(y=C_1e^x+C_2e^{6x}+\frac{7}{74}\cos x+\frac{5}{74}\sin x\)
(6)\(\displaystyle 2y''+5y'=5x^{2}-2x-1\)

齐次通解 \(y_h=C_1\cos 2\sqrt{2}x+C_2\sin 2\sqrt{2}x\)
设特解 \(y_p=A\sin 2x+B\cos 2x\)
代入 \(y_p''=-4A\sin 2x-4B\cos 2x\)，得 \(-4A\sin 2x-4B\cos 2x+8A\sin 2x+8B\cos 2x=8\sin 2x\)
即 \(4A\sin 2x+4B\cos 2x=8\sin 2x\)，比较系数得 \(A=2, B=0\)
特解 \(y_p=2\sin 2x\)
通解 \(y=C_1\cos 2\sqrt{2}x+C_2\sin 2\sqrt{2}x+2\sin 2x\)
由 \(y(0)=0\)：\(C_1+0+0=0\)，得 \(C_1=0\)
由 \(y\left(\frac{\pi}{2}\right)=0\)：\(C_2\sin \sqrt{2}\pi+2\sin \pi=0\)，即 \(C_2\sin \sqrt{2}\pi=0\)
因 \(\sin \sqrt{2}\pi\neq 0\)（\(\sqrt{2}\) 为无理数），故 \(C_2=0\)
特解为 \(y=2\sin 2x\)
答案\(y=2\sin 2x\)
